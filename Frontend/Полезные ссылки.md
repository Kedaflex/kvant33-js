# 🔗 Полезные ссылки по веб-разработке

Здесь собраны дополнительные ресурсы для углубленного изучения тем курса.

## 🧱 CSS Блочная модель (Модуль 7)

- [MDN Web Docs: The box model](https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/The_box_model) — Базовое руководство от MDN.
- [CSS Tricks: Box-sizing](https://css-tricks.com/box-sizing/) — Всё о свойстве box-sizing.
- [MDN Web Docs: Mastering margin collapsing](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Box_Model/Mastering_margin_collapsing) — Подробно о схлопывании отступов.
- [Interactive Guide to Flexbox](https://www.joshwcomeau.com/css/interactive-guide-to-flexbox/) — (Для будущих модулей, но полезно знать).
- [Can I Use](https://caniuse.com/) — Проверка поддержки свойств браузерами.

## 📍 Позиционирование и слои (Модуль 8)

- [MDN Web Docs: CSS Layout - Positioning](https://developer.mozilla.org/en-US/docs/Learn/CSS/CSS_layout/Positioning) — Подробный гайд по позиционированию.
- [MDN Web Docs: The stacking context](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Positioning/Understanding_z_index/The_stacking_context) — Глубокое погружение в z-index.
- [Z-Index Explained](https://www.joshwcomeau.com/css/stacking-contexts/) — Интерактивное объяснение контекста наложения (англ.).
- [CSS-Tricks: Pseudo-elements](https://css-tricks.com/almanac/selectors/a/after-and-before/) — Всё о ::before и ::after.
- [Nth-child Tester](https://nth-test.com/) — Визуальный тестер для сложных селекторов :nth-child.
