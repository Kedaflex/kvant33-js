# 🎯 justify-content: Выравнивание по главной оси

## 📖 Что делает justify-content?

Когда мы размещаем элементы внутри Flex-контейнера, они часто занимают не всю доступную ширину (или высоту). Оставшееся пустое место называется **«свободным пространством»**.

Свойство `justify-content` определяет, как браузер должен распорядиться этим пространством. Должен ли он оставить его в конце? В начале? Или распределить между элементами?

**Важнейшее правило:** `justify-content` работает **всегда** вдоль **Главной оси**.
- Если `flex-direction: row`, то свойство управляет горизонтальным выравниванием.
- Если `flex-direction: column`, то свойство управляет вертикальным выравниванием.

---

## 🛠️ Основные значения

### 1. flex-start (по умолчанию)
Элементы прижаты к началу главной оси (обычно к левому краю). Свободное пространство остается в конце.
```css
justify-content: flex-start;
```

### 2. flex-end
Элементы прижаты к концу главной оси (обычно к правому краю). Свободное пространство остается в начале.
```css
justify-content: flex-end;
```
*Заметьте: в отличие от `row-reverse`, порядок элементов не меняется, они просто «переезжают» в другой конец контейнера.*

### 3. center
Элементы собираются в центре. Свободное пространство делится поровну и распределяется по краям. Это самый простой и надежный способ горизонтального центрирования в истории CSS.
```css
justify-content: center;
```

---

## 🌌 Значения для распределения пространства

Эти значения наиболее интересны, так как они позволяют создавать профессиональные сетки без ручного расчета маргинов.

### 4. space-between
Первый элемент прижимается к началу, последний — к концу. Всё свободное пространство делится на равные части и вставляется **между** элементами.
*Идеально для навигационных панелей (логотип слева — меню справа).*

### 5. space-around
Свободное пространство распределяется так, что вокруг каждого элемента есть одинаковые отступы.
*Нюанс:* Расстояние между двумя соседними элементами будет в два раза больше, чем расстояние между краем контейнера и крайним элементом (потому что отступы двух соседних элементов складываются).

### 6. space-evenly
Свободное пространство распределяется так, что расстояние между любыми двумя элементами, а также расстояние до краев контейнера, будет **абсолютно одинаковым**.
*Самый современный и визуально гармоничный способ распределения.*

---

## 🔄 justify-content в режиме колонок (column)

Это тот момент, когда нужно включить пространственное мышление. Если вы задали `flex-direction: column`, то ваш контейнер стал вертикальным стеком.

Теперь `justify-content: center` будет центрировать элементы **по вертикали**.
**Важно:** Чтобы это сработало, у контейнера должна быть задана высота (`height`). Если высота контейнера равна высоте суммы элементов внутри него, вы не увидите изменений, так как свободного пространства просто нет!

## 💡 Секреты и нюансы

### Магия center
Многие новички пытаются центрировать всё подряд. Помните, что `justify-content` центрирует **всю группу** элементов как единое целое. Если вы хотите, чтобы один элемент был слева, а другой справа — используйте `space-between`.

### Проблема flex-start/end vs start/end
В современных браузерах появились значения `start` и `end` (без приставки `flex-`). Они более универсальны для Grid и Flexbox, но для максимальной поддержки старых устройств все еще рекомендуется использовать `flex-start` и `flex-end`.

### Почему выравнивание не работает?
Если вы написали `justify-content: center`, но элементы не сдвинулись:
1. Проверьте, задан ли `display: flex` родителю.
2. Проверьте ширину (или высоту в режиме колонок) контейнера. Если контейнер «схлопнулся» по размеру детей, выравнивать нечего.
3. Проверьте, нет ли у элементов `flex-grow: 1`. Если элементы растянулись на всю ширину, свободного места не осталось.

## ❓ Часто задаваемые вопросы

### «Что лучше: space-around или space-evenly?»
Это зависит от дизайна. `space-around` делает акцент на самих элементах, давая им «воздух». `space-evenly` делает акцент на ритме сетки, создавая математически точные промежутки. Чаще всего в современном дизайне предпочитают `space-evenly` или `gap` (который мы изучим позже).

### «Как прижать только один элемент в конец?»
Если вам нужно, чтобы три ссылки были слева, а кнопка — справа, не используйте `justify-content`. Используйте `justify-content: flex-start` для всех, а кнопке задайте `margin-left: auto`. Это «чит-код» Flexbox, который часто работает лучше любого выравнивания.

## 🏗️ Практика: Панель инструментов (Toolbar)

Создадим типичный элемент интерфейса, используя разные типы выравнивания.

```html
<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <style>
    .toolbar {
      display: flex;
      background: #2c3e50;
      padding: 10px 20px;
      color: white;
      align-items: center; /* Выравнивание по вертикали, изучим в след. уроке */
      margin-bottom: 20px;
    }

    /* Попробуйте по очереди менять значения здесь: */
    .between { justify-content: space-between; }
    .around  { justify-content: space-around; }
    .center  { justify-content: center; }
    .evenly  { justify-content: space-evenly; }

    .btn {
      background: #3498db;
      border: none;
      padding: 8px 15px;
      color: white;
      border-radius: 4px;
      cursor: pointer;
    }

    .logo { font-weight: bold; font-size: 20px; }
  </style>
</head>
<body>

  <h3>Space-Between (Лого + Меню)</h3>
  <div class="toolbar between">
    <div class="logo">MyLogo</div>
    <div class="menu">
      <button class="btn">Главная</button>
      <button class="btn">О нас</button>
    </div>
  </div>

  <h3>Space-Evenly (Равномерные кнопки)</h3>
  <div class="toolbar evenly">
    <button class="btn">Action 1</button>
    <button class="btn">Action 2</button>
    <button class="btn">Action 3</button>
  </div>

</body>
</html>
```

## 🔑 Ключевые мысли урока

> [!abstract] Запомни
> 1. `justify-content` управляет распределением **свободного места**.
> 2. Работает исключительно по **Главной оси**.
> 3. `center` — самый простой способ центрирования.
> 4. `space-between` прижимает крайние элементы к границам контейнера.
> 5. `space-evenly` гарантирует абсолютно равные промежутки везде.
> 6. Если выравнивание не работает — проверьте наличие свободного места в контейнере.
