## 📖 Прежде чем начать

В прошлом уроке ты создал папку `web-course` и внутри неё папку `001-hello-world`. Если не сделал — вернись к уроку [[004 — Установка VS Code]] и выполни задание.

Сейчас мы впервые **напишем HTML-код** и увидим результат в браузере. Это будет твой первый «Hello, World!» — традиция, с которой программисты начинают изучение любой технологии.

> [!tip] Традиция «Hello, World!»
> Первую программу «Hello, World!» написал Брайан Керниган в 1972 году в книге по языку C. С тех пор каждый программист начинает с этой фразы. Теперь твоя очередь.

## 🛠️ Шаг 1: Создаём файл

| Шаг | Действие |
|---|---|
| 1 | Открой VS Code |
| 2 | Открой папку `web-course` через **File → Open Folder** |
| 3 | В боковой панели найди папку `001-hello-world` |
| 4 | Наведи на неё курсор — появятся иконки |
| 5 | Нажми иконку 📄 (New File) |
| 6 | Назови файл `index.html` |
| 7 | Нажми Enter |

Файл создан и открыт в редакторе. Пока он пустой.

> [!note] Почему `index.html`?
> Файл `index.html` — это **главная страница** сайта по умолчанию. Когда ты заходишь на `google.com`, сервер отдаёт именно файл `index.html`. Это общепринятое соглашение. Мы будем называть главную страницу каждого проекта `index.html`.

---

## 🛠️ Шаг 2: Пишем первый HTML-код

Набери в файле `index.html` следующий код **вручную** (не копируй — так ты лучше запомнишь):

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Моя первая страница</title>
  </head>
  <body>
    <h1>Привет, мир!</h1>
    <p>Это моя первая HTML-страница.</p>
  </body>
</html>
```

> [!warning] Набирай вручную!  
> Очень важно **печатать код руками**, а не копировать. При ручном наборе ты запоминаешь синтаксис в разы быстрее. Пальцы привыкают к угловым скобкам, слешам и кавычкам.

---
## 🛠️ Шаг 3: Сохраняем файл

Нажми `Ctrl + S` (или `Cmd + S` на Mac).

Обрати внимание на название файла во вкладке:

|Индикатор|Значение|
|---|---|
|`index.html`|Файл сохранён|
|`index.html ●` (с точкой)|Файл **не сохранён**, есть несохранённые изменения|

> [!tip] Привычка сохранять  
> Приучи себя нажимать `Ctrl + S` после каждого изменения. Со временем это станет рефлексом — пальцы сами будут нажимать сохранение.

## 🛠️ Шаг 4: Открываем в браузере

### Способ 1: Live Server (рекомендуется)

|Шаг|Действие|
|---|---|
|1|Кликни правой кнопкой мыши в области редактора|
|2|Выбери **«Open with Live Server»**|
|3|Откроется браузер с твоей страницей|

Или нажми кнопку **«Go Live»** в правом нижнем углу VS Code.

### Способ 2: Просто открыть файл

|Шаг|Действие|
|---|---|
|1|Открой **Проводник** (или Finder)|
|2|Найди файл `index.html` в папке `001-hello-world`|
|3|Дважды кликни по файлу|
|4|Файл откроется в браузере|

> [!note] Разница между способами
> 
> |Способ|Автообновление|Адрес в браузере|
> |---|---|---|
> |**Live Server**|✅ Да, обновляется при сохранении|`http://127.0.0.1:5500/...`|
> |**Открыть файл**|❌ Нет, нужно обновлять вручную (F5)|`file:///C:/Users/...`|
> 
> Live Server удобнее. Используй его.

## 🔍 Разберём код построчно

Пока не нужно заучивать каждую строку. Просто пойми общую логику — в следующем уроке мы разберём структуру подробнее.

|Строка|Код|Что делает|
|---|---|---|
|1|`<!DOCTYPE html>`|Говорит браузеру: «Это HTML5 документ»|
|2|`<html>`|Начало HTML-документа|
|3|`<head>`|Начало служебной информации (не видна на странице)|
|4|`<title>Моя первая страница</title>`|Заголовок вкладки в браузере|
|5|`</head>`|Конец служебной информации|
|6|`<body>`|Начало видимой части страницы|
|7|`<h1>Привет, мир!</h1>`|Главный заголовок страницы|
|8|`<p>Это моя первая HTML-страница.</p>`|Абзац текста|
|9|`</body>`|Конец видимой части страницы|
|10|`</html>`|Конец HTML-документа|

## 🏷️ Что такое теги (первое знакомство)

Ты заметил, что весь код состоит из слов в **угловых скобках** `< >`. Это и есть **теги**.

**Тег** — это команда для браузера. Он говорит: «Вот этот кусок текста — заголовок», «А вот этот — абзац».

### Два вида тегов

|Вид|Описание|Пример|
|---|---|---|
|**Парный тег**|Имеет открывающую и закрывающую часть|`<h1>текст</h1>`|
|**Одиночный тег**|Не имеет закрывающей части|`<br>`, `<img>` (изучим позже)|

### Анатомия парного тега

Разберём тег `<h1>Привет, мир!</h1>`:

|Часть|Код|Название|
|---|---|---|
|Открывающий тег|`<h1>`|Говорит: «Здесь начинается заголовок»|
|Содержимое|`Привет, мир!`|Текст, который увидит пользователь|
|Закрывающий тег|`</h1>`|Говорит: «Здесь заголовок заканчивается»|

> [!important] Слеш в закрывающем теге  
> Обрати внимание: закрывающий тег отличается от открывающего **символом `/`** (слеш). Открывающий: `<h1>`. Закрывающий: `</h1>`. Если забыть слеш — браузер не поймёт, где заканчивается элемент.

## 🧪 Эксперимент 1: Добавляем больше контента

Давай добавим на страницу ещё элементов. Замени содержимое `<body>` на:



```HTML
<body>
    <h1>Привет, мир!</h1>
    <p>Это моя первая HTML-страница.</p>
    <h2>Обо мне</h2>
    <p>Меня зовут Алексей. Я учу веб-разработку.</p>
    <h2>Мои цели</h2>
    <p>Я хочу научиться создавать сайты.</p>
    <p>Это будет полезный навык.</p>
</body>
```

Сохрани (`Ctrl + S`) и посмотри результат в браузере.

### Что нового?

|Тег|Что делает|
|---|---|
|`<h1>`|Заголовок **первого уровня** (самый крупный, самый главный)|
|`<h2>`|Заголовок **второго уровня** (поменьше, подзаголовок)|
|`<p>`|**Абзац** текста (paragraph)|

> [!note] Заголовки h1–h6  
> В HTML есть 6 уровней заголовков: от `<h1>` (самый важный) до `<h6>` (наименее важный). Мы подробно разберём их в уроке [[009 — Заголовки h1–h6]].

## 🧪 Эксперимент 2: Добавляем список

Добавь после последнего `<p>` следующий код:



```HTML
<h2>Что я буду изучать</h2>
<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ul>
```

Сохрани и посмотри результат.

### Что нового?

|Тег|Что делает|
|---|---|
|`<ul>`|Начало **маркированного списка** (unordered list)|
|`<li>`|**Элемент списка** (list item)|
|`</ul>`|Конец списка|

В браузере ты увидишь список с точками (маркерами):

- HTML
- CSS
- JavaScript

## 🧪 Эксперимент 3: Добавляем ссылку

Добавь после списка:



```HTML
<p>
    Вот полезный сайт для обучения:
    <a href="https://developer.mozilla.org">MDN Web Docs</a>
</p>
```

Сохрани и посмотри результат. На странице появится кликабельная ссылка.

### Что нового?

|Тег|Что делает|
|---|---|
|`<a>`|**Ссылка** (anchor)|
|`href="..."`|**Атрибут** — указывает, куда ведёт ссылка|

> [!note] Атрибуты  
> `href` — это **атрибут** тега `<a>`. Атрибуты добавляют тегам дополнительную информацию. Мы подробно разберём атрибуты в уроке [[008 — Атрибуты тегов]].

## 🧪 Эксперимент 4: Жирный и курсивный текст

Замени один из абзацев на:



```HTML
<p>Я <strong>очень хочу</strong> научиться <em>веб-разработке</em>.</p>
```

Сохрани и посмотри результат.

### Что нового?

|Тег|Что делает|Как выглядит|
|---|---|---|
|`<strong>`|**Важный** текст|**Жирный**|
|`<em>`|**Акцентированный** текст|_Курсивный_|

## ❓ Частые вопросы

### «Я набрал код, но в браузере ничего не появляется»

|Проблема|Решение|
|---|---|
|Файл не сохранён|Нажми `Ctrl + S`. Проверь, нет ли точки ● на вкладке|
|Файл не `.html`|Убедись, что файл называется `index.html`, а не `index.txt`|
|Открыл не тот файл|Проверь путь в адресной строке браузера|
|Опечатка в коде|Сверь свой код с примером выше, символ за символом|

### «Страница выглядит некрасиво, просто чёрный текст на белом фоне»

Это **нормально**. Сейчас мы написали только HTML — **структуру**. Красоту добавляет CSS, который мы начнём изучать в модуле 5. Пока наша задача — научиться правильно размечать контент.

### «VS Code подсвечивает код разными цветами — это нормально?»

Да! Это **подсветка синтаксиса**. Она помогает различать части кода:

|Цвет (в стандартной тёмной теме)|Что обозначает|
|---|---|
|Синий / красный|Названия тегов (`h1`, `p`, `a`)|
|Светло-зелёный|Атрибуты (`href`)|
|Оранжевый / жёлтый|Значения атрибутов (`"https://..."`)|
|Белый / серый|Текстовое содержимое|

### «Можно ли писать теги БОЛЬШИМИ буквами? `<H1>` вместо `<h1>`?»

Технически — да, браузер поймёт. Но принято писать **маленькими буквами**. Это стандарт и хорошая практика. Всегда пиши теги в нижнем регистре.

## 🔑 Ключевые мысли урока

> [!abstract] Запомни
> 
> 1. HTML-файл — это обычный текстовый файл с расширением `.html`
> 2. Главная страница сайта по соглашению называется `index.html`
> 3. **Тег** — это команда для браузера в угловых скобках `< >`
> 4. Парные теги имеют **открывающую** `<h1>` и **закрывающую** `</h1>` часть
> 5. Всё, что внутри `<body>`, — **видно** на странице
> 6. Всё, что внутри `<head>`, — **не видно**, это служебная информация
> 7. Сохраняй файл (`Ctrl + S`) после каждого изменения
