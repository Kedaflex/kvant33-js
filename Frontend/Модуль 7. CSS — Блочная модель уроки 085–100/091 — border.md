# 🖼️ Границы элементов: Свойство border

← [[090 — margin]] | [[092 — box-sizing]] →

> [!info] Цели урока
> - Научиться рисовать рамки вокруг элементов.
> - Разобраться в синтаксисе свойства `border`.
> - Узнать, как скруглять углы с помощью `border-radius`.

---

## 📖 Введение: Очерчиваем границы

До сих пор наши блоки были либо прозрачными, либо просто залиты цветом. Но что если мы хотим выделить кнопку, отделить одну карточку от другой или создать стильную рамку для фотографии?

Свойство `border` — это «контур» вашего элемента. Оно находится ровно между `padding` (внутренним отступом) и `margin` (внешним отступом).

Рамка — это не просто украшение. Это важный визуальный сигнал для пользователя, который помогает понять, где заканчивается один элемент и начинается другой.

---

## 🛠️ Три кита рамки: Ширина, Стиль, Цвет

Для того чтобы рамка появилась, браузеру нужно знать три параметра. Если вы забудете хотя бы один (особенно стиль), рамка скорее всего не отобразится.

### 1. border-width (Ширина)
Определяет толщину линии. Можно задавать в пикселях или ключевыми словами: `thin`, `medium`, `thick`.
```css
.box { border-width: 5px; }
```

### 2. border-style (Стиль) — Самое важное!
Определяет вид линии. По умолчанию стоит `none`, поэтому рамки не видно.
- `solid` — обычная сплошная линия (самый частый выбор).
- `dashed` — пунктирная (тире).
- `dotted` — точечная.
- `double` — двойная линия.

### 3. border-color (Цвет)
Любой доступный в CSS формат цвета (HEX, RGB, название). Если не указать, браузер возьмет текущий цвет текста элемента.

---

## 🚀 Сокращенная запись (Shorthand)

Вместо того чтобы писать три свойства, мы обычно используем одну короткую строку:
`border: [ширина] [стиль] [цвет];`

```css
.card {
  border: 2px solid #333;
}
```
*Порядок значений в сокращенной записи не важен, но общепринято писать именно так.*

### Отдельные стороны
Вы можете задать рамку только для одной стороны. Это часто используется для подчеркивания или создания декоративных разделителей.
```css
.header {
  border-bottom: 1px solid #ccc; /* Тонкая линия снизу */
}
.sidebar {
  border-left: 5px solid blue; /* Жирная полоса слева */
}
```

## 🟠 Скругление углов: border-radius

Хотя это свойство технически не относится к «рамке» (оно работает, даже если рамки нет), они всегда идут рука об руку.

`border-radius` позволяет превратить острые углы прямоугольника в скругленные.

```css
.button {
  border-radius: 8px; /* Легкое скругление */
}

.avatar {
  width: 100px;
  height: 100px;
  border-radius: 50%; /* Превращает квадрат в идеальный круг */
}
```

**Нюансы border-radius:**
1. Можно задавать разные радиусы для каждого угла: `border-radius: 10px 20px 30px 40px;` (Верх-Лево, Верх-Право, Низ-Право, Низ-Лево).
2. Если вы зададите радиус больше половины стороны квадрата, он перестанет меняться (останется кругом).
3. Фон элемента (`background`) тоже обрезается по этим скругленным углам.

---

## 📏 Border и размер блока

Как и `padding`, рамка **увеличивает** размер элемента.
Если вы задали `width: 100px` и `border: 10px solid black`, на экране элемент будет занимать **120px** (100 + 10 слева + 10 справа).

> [!warning] Важно
> Рамка всегда занимает место. Если вы добавите рамку при наведении (`:hover`), ваш контент может «дернуться», так как блок внезапно станет на пару пикселей шире. Чтобы этого избежать, используют либо прозрачную рамку по умолчанию, либо свойство `outline`.

---

## 🔍 Креативные приемы с border

### 1. Создание треугольников
Знаете ли вы, что с помощью `border` можно рисовать треугольники без картинок? Если задать элементу нулевую ширину и высоту, но очень толстые рамки разных цветов, вы увидите, как они стыкуются под углом 45 градусов. Сделав три стороны прозрачными, вы получите треугольник.

### 2. Рисование разделителей
Вместо тега `<hr>` часто лучше использовать `border-top` у следующего блока. Это дает больше контроля над отступами.

## ❓ Часто задаваемые вопросы

### «Почему `border: 1px solid red` выглядит размыто на моем экране?»
На экранах с высокой плотностью пикселей (Retina) 1px — это очень мало. Иногда браузеры пытаются сгладить такие линии. Попробуйте использовать `0.5px` (в современных браузерах) или просто смиритесь — это особенность рендеринга.

### «В чем разница между border и outline?»
- `border` является частью блочной модели и **занимает место**.
- `outline` рисуется **поверх** элемента, не занимает места и не влияет на положение соседей. Обычно используется для выделения элементов при навигации с клавиатуры (фокус).

### «Можно ли сделать градиентную рамку?»
Напрямую через `border-color` — нет. Но можно использовать свойство `border-image`, которое позволяет использовать картинку или градиент в качестве рамки. Это более сложная тема.

## 🧪 Практика: Карточка со стилем

Давайте соберем знания воедино и создадим красивый элемент.

```html
<!DOCTYPE html>
<html lang="ru">
<head>
  <style>
    .profile-card {
      width: 300px;
      margin: 50px auto;
      text-align: center;
      padding: 20px;

      /* Работа с границами */
      border: 1px solid #ddd;
      border-top: 5px solid #3498db; /* Акцент сверху */
      border-radius: 15px;

      background: #fff;
      box-shadow: 0 4px 6px rgba(0,0,0,0.1); /* Тень для объема */
    }

    .avatar {
      width: 80px;
      height: 80px;
      background: #eee;
      margin: 0 auto 15px;

      /* Делаем круг */
      border-radius: 50%;
      border: 3px solid #3498db;
    }

    h3 {
      margin: 0;
      color: #333;
    }
  </style>
</head>
<body>

  <div class="profile-card">
    <div class="avatar"></div>
    <h3>Иван Иванов</h3>
    <p>Веб-разработчик</p>
  </div>

</body>
</html>
```
