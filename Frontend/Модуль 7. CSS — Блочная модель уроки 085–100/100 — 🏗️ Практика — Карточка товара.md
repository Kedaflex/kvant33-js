# 🏗️ Практика: Создаем современную карточку товара

> - Использовать продвинутые селекторы для точечной стилизации элементов.
> - Создать законченный компонент в стиле современного минимализма.

## 📖 Введение: От теории к макету

Мы прошли огромный путь в этом модуле. Мы узнали, как CSS превращает скучный HTML-текст в набор гибких и красивых блоков. Теперь настало время «финального босса» этого модуля.

Сегодня мы не будем просто изучать свойства — мы будем строить. Наша цель — создать карточку товара для интернет-магазина. Она должна быть аккуратной, адаптивной и приятной на ощупь (визуально).

В этом проекте мы применим:
1. `display: inline-block` для размещения карточек в ряд.
2. `box-sizing: border-box` для удобного расчета размеров.
3. `margin: auto` для центрирования контента.
4. `box-shadow` и `border-radius` для современного вида.
5. Селекторы потомков и соседей для чистого кода.

---

## 📐 План работы

1. **Разметка**: Подготовим HTML-структуру (контейнер, изображение, текст, кнопка).
2. **Сброс**: Подключим базовые настройки (`border-box`).
3. **Основа**: Сформируем размеры и фон самой карточки.
4. **Контент**: Настроим отступы и типографику внутри.
5. **Акценты**: Добавим тени и эффекты при наведении.

---

## 🛠️ Шаг 1: HTML-структура

Используем семантическую разметку. Каждая карточка — это самодостаточный блок.

```html
<div class="product-card">
  <div class="product-badge">Новинка</div>
  <div class="product-image">
    <img src="https://via.placeholder.com/260x200" alt="Беспроводные наушники">
  </div>
  <div class="product-info">
    <h3 class="product-title">Беспроводные наушники Pro</h3>
    <p class="product-description">Чистый звук, активное шумоподавление и до 30 часов работы.</p>
    <div class="product-price">
      <span class="current-price">12 990 ₽</span>
      <span class="old-price">15 990 ₽</span>
    </div>
    <button class="product-button">В корзину</button>
  </div>
</div>
```

---

## 🎨 Шаг 2: Базовые стили и сброс

Начинаем с фундамента.

```css
/* Глобальный сброс — теперь мы повелители размеров */
*, *::before, *::after {
  box-sizing: border-box;
}

body {
  font-family: 'Inter', sans-serif;
  background-color: #f5f5f5;
  padding: 50px;
  display: flex;
  justify-content: center;
  gap: 20px;
}
```

---

## 📦 Шаг 3: Стилизация контейнера карточки

Сделаем карточку «мягкой» и объемной.

```css
.product-card {
  width: 300px;
  background: white;
  border-radius: 20px;
  overflow: hidden; /* Чтобы картинка не вылезала за скругленные углы */
  position: relative;
  transition: transform 0.3s ease, box-shadow 0.3s ease;
  border: 1px solid #eee;
}

.product-card:hover {
  transform: translateY(-8px);
  box-shadow: 0 20px 40px rgba(0,0,0,0.08);
}
```

---

## 🖼️ Шаг 4: Изображение и бейдж

Научим картинку вести себя правильно.

```css
.product-image {
  background-color: #f9f9f9;
  height: 200px;
  display: flex;
  align-items: center;
  justify-content: center;
}

.product-image img {
  max-width: 100%;
  height: auto;
}

.product-badge {
  position: absolute;
  top: 15px;
  left: 15px;
  background-color: #000;
  color: #fff;
  padding: 5px 12px;
  font-size: 12px;
  font-weight: bold;
  border-radius: 50px;
  z-index: 10;
}
```

---

## 📝 Шаг 5: Типографика и отступы

Используем `padding` для создания «воздуха» и селекторы соседей для управления ритмом.

```css
.product-info {
  padding: 24px;
}

.product-title {
  margin: 0 0 10px 0;
  font-size: 18px;
  color: #1a1a1a;
}

.product-description {
  margin: 0 0 20px 0;
  font-size: 14px;
  color: #666;
  line-height: 1.5;
}

.product-price {
  margin-bottom: 24px;
}

.current-price {
  font-size: 20px;
  font-weight: bold;
  color: #000;
  margin-right: 10px;
}

.old-price {
  font-size: 14px;
  color: #999;
  text-decoration: line-through;
}
```

---

## 🛒 Шаг 6: Интерактивная кнопка

Завершаем образ стильной кнопкой.

```css
.product-button {
  width: 100%;
  padding: 14px;
  background-color: #000;
  color: #fff;
  border: none;
  border-radius: 12px;
  font-weight: 600;
  cursor: pointer;
  transition: background 0.2s;
}

.product-button:hover {
  background-color: #333;
}

.product-button:active {
  transform: scale(0.98);
}
```

## 🔍 Анализ проделанной работы

Давайте посмотрим, как наши знания сработали здесь:
1. **border-box**: Позволил нам задать `width: 100%` кнопке и добавить ей `padding`, не боясь, что она вылезет за края карточки.
2. **padding: 24px**: Создал равномерный внутренний отступ для всего текстового блока.
3. **overflow: hidden**: Спас нас от того, что прямоугольная картинка сверху перекрывала скругленные углы родительской карточки.
4. **margin-bottom**: У каждого текстового элемента свой отступ вниз, что создает четкую иерархию.
5. **transition**: Сделал взаимодействие с карточкой «дорогим» и плавным.

## ❓ Часто задаваемые вопросы

### «Почему бы не использовать Flexbox для всей карточки?»
Мы сделаем это в Модуле 9! Но очень важно уметь верстать такие компоненты на базовой блочной модели. Это дает глубокое понимание того, как элементы взаимодействуют друг с другом «от природы».

### «Как сделать так, чтобы все карточки в ряду были одной высоты?»
На чистой блочной модели это сделать сложно (придется задавать фиксированный `height`). Именно поэтому для сеток чаще используют Flexbox или Grid, которые мы изучим совсем скоро.

### «Почему мы использовали чёрный цвет для кнопки?»
В современном минимализме часто используют чистый чёрный (#000) или очень тёмно-серый (#1a1a1a) для создания сильного контраста на белом фоне. Это выглядит стильно и профессионально.

## 🔑 Ключевые мысли урока

> [!abstract] Запомни
> 1. Начинайте верстку с **HTML-структуры**, а не со стилей.
> 2. Используйте **отступы (padding)**, чтобы контент не задыхался.
> 3. **Группируйте** связанные элементы (например, цену и старую цену) в отдельные контейнеры.
> 4. **Интерактивность** (hover, active) делает интерфейс живым.
> 5. **box-sizing: border-box** — обязательная настройка для любого современного компонента.
