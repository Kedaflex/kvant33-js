## 📖 Введение: Масштабируем красоту

В прошлом уроке мы научились вставлять картинки на фон. Но часто возникает проблема: картинка либо слишком маленькая и оставляет пустое место, либо слишком большая и мы видим только её крошечный уголок.

В современном вебе, где размеры экранов постоянно меняются, нам нужны инструменты, которые позволят картинке «дышать» и подстраиваться под размер блока. Именно для этого существуют свойства `background-size` (размер) и `background-position` (положение).

---

## 📏 1. Размер фона: `background-size`

Это свойство определяет, как картинка должна масштабироваться внутри элемента.

### Ключевые слова-спасатели:
1. **`cover`** (Заполнить): Самое популярное значение. Картинка растягивается так, чтобы **полностью закрыть** весь блок. Если пропорции картинки и блока не совпадают, лишние края картинки будут просто обрезаны. Пустых мест не останется.
2. **`contain`** (Вместить): Картинка масштабируется так, чтобы **целиком поместиться** в блоке. Она не будет обрезана, но если блок шире картинки, по бокам или сверху могут остаться пустые поля.

### Точные значения:
Вы можете задать размер в пикселях или процентах.
- `background-size: 100% 100%;` — Растянуть картинку ровно по размеру блока (внимание: пропорции могут исказиться, картинка станет «сплющенной»).
- `background-size: 300px auto;` — Ширина 300 пикселей, высота подберется автоматически для сохранения пропорций.

```css
.hero {
  background-image: url('landscape.jpg');
  background-size: cover; /* Картинка на весь экран без дырок */
}
```

## ⚓ 2. Позиция фона: `background-position`

Если картинка больше блока и обрезается (например, при `cover`), вы должны решить, какую именно часть картинки мы хотим видеть: центр, верхний край или, может быть, правый угол?

### Использование ключевых слов:
По умолчанию фон стоит в левом верхнем углу (`top left`). Вы можете комбинировать слова:
- `center center` (или просто `center`): Поставить центр картинки в центр блока.
- `top right`: Прижать к верхнему правому углу.
- `bottom left`: Прижать к нижнему левому углу.

### Использование процентов и пикселей:
Для более точной настройки:
- `background-position: 50% 50%;` — То же самое, что `center`.
- `background-position: 20px 100px;` — Сдвинуть картинку на 20px вправо и 100px вниз от левого верхнего угла.

```css
.profile-bg {
  background-image: url('person.jpg');
  background-size: cover;
  background-position: center top; /* Всегда показываем голову, даже если ноги обрежутся */
}
```

## 🚀 Секрет идеального фонового изображения

Когда вы создаете «Герой-секцию» (Hero Section) вверху сайта, профессиональный стандарт выглядит так:

```css
.hero {
  min-height: 100vh; /* Секция на всю высоту экрана */
  background-image: url('epic-photo.jpg');
  background-repeat: no-repeat;
  background-position: center; /* Центрируем самое важное */
  background-size: cover; /* Растягиваем на весь экран */
  background-color: #333; /* Запасной цвет */
}
```

Эта комбинация гарантирует, что на любом устройстве — от смартфона до телевизора — ваша картинка будет выглядеть адекватно.

## 📐 3. Начало отсчета: `background-origin`

Это свойство определяет, откуда именно начинает рисоваться фон:
- **`padding-box`**: От края внутренних отступов (по умолчанию).
- **`border-box`**: Фон заходит даже под рамку (границу) элемента.
- **`content-box`**: Фон рисуется только там, где есть текст или контент.

Это используется редко, но полезно, если вы делаете сложные дизайнерские рамки.

---

## ✂️ 4. Обрезка фона: `background-clip`

Похоже на `origin`, но работает наоборот — оно обрезает фон.
Особо интересное значение — **`text`**. Оно позволяет «залить» текст картинкой!

```css
.text-with-bg {
  background-image: url('gold.jpg');
  -webkit-background-clip: text; /* Обрезаем фон по форме букв */
  color: transparent; /* Делаем сами буквы прозрачными */
}
```

## 🧪 Практика: Карточка-превью с "умным" фоном

Давайте создадим элемент, в котором картинка всегда будет по центру, как бы мы ни меняли размер блока.

```html
<div class="photo-card">
  <div class="photo-overlay">
    <h3>Горы Кавказа</h3>
  </div>
</div>
```

```css
.photo-card {
  width: 300px;
  height: 400px;
  border: 5px solid white;
  box-shadow: 0 4px 15px rgba(0,0,0,0.2);

  /* Настройка фона */
  background-image: url('https://images.unsplash.com/photo-1464822759023-fed622ff2c3b');
  background-repeat: no-repeat;
  background-size: cover;
  background-position: center;

  /* Центрируем текст внутри */
  display: flex;
  align-items: flex-end;
  color: white;
  font-family: sans-serif;
}

.photo-overlay {
  background: rgba(0,0,0,0.5);
  width: 100%;
  padding: 10px;
}
```

## ⚠️ Типичные ошибки

1. **Искажение пропорций**: Использование `background-size: 100% 100%`. Это заставляет картинку неестественно растягиваться. Всегда используйте `cover` или `auto`.
2. **Слишком тяжелые картинки**: Использование `cover` не уменьшает вес файла. Если вы загрузили фото 10МБ, оно так и будет весить 10МБ, даже если вы сжали его до размера 100px через CSS.
3. **Обрезка важных деталей**: Не использовать `background-position`. Например, на фото человек стоит справа, а блок узкий. Без позиционирования браузер покажет пустую левую часть фото, а человека обрежет.
4. **Забытый `no-repeat`**: При использовании `cover` повторение обычно не видно, но если картинка не прогрузится до конца, вы можете увидеть некрасивые стыки.

## ❓ Часто задаваемые вопросы

### «В чем разница между cover и 100%?»
`100%` растягивает картинку ровно до границ блока. Если блок широкий и низкий, а картинка квадратная — она станет плоской. `cover` же сохраняет пропорции картинки, просто отрезая то, что не влезло.

### «Можно ли анимировать позицию фона?»
Да! Вы можете сделать так, чтобы при наведении мыши фон плавно «уплывал» в сторону. Это создаёт очень красивый эффект глубины.

### «Как сделать так, чтобы фон менялся в зависимости от размера экрана?»
Для этого используются медиа-запросы (`@media`), которые мы изучим в модуле про адаптивность. Например, для телефонов можно подгружать вертикальную картинку, а для ПК — горизонтальную.

## 🔑 Ключевые мысли урока

> [!abstract] Запомни
> 1. **`background-size: cover`** — лучший друг для полноэкранных фонов.
> 2. **`background-size: contain`** — если нужно показать картинку целиком без обрезки.
> 3. **`background-position`** управляет тем, какой участок картинки мы видим.
> 4. Ключевые слова `center`, `top`, `bottom` — самые надежные для позиционирования.
> 5. Всегда проверяйте, как ваш фон выглядит при изменении размера окна браузера.
> 6. Грамотное использование этих свойств делает сайт профессиональным и адаптивным.
