## 📖 Введение: Мир миллионов оттенков

Цвет — это мощнейший инструмент влияния на пользователя. С его помощью мы можем задать настроение сайта (спокойное синее, агрессивное красное или чистое белое), выделить важные кнопки и создать визуальную иерархию.

В CSS существует несколько систем записи цвета. Каждая из них была придумана для решения своих задач: от простоты написания до математической точности. В этом уроке мы разберем их все, чтобы вы чувствовали себя уверенно при работе с любым макетом дизайнера.

---

## 🎨 1. Именованные цвета (Keywords)

Это самый простой способ. Вы просто пишете название цвета на английском языке.

```css
h1 { color: red; }
p { background-color: skyblue; }
```

- **Количество**: Всего существует около 147 стандартных имен (например, `aliceblue`, `darkgoldenrod`, `tomato`).
- **Плюсы**: Легко запомнить, быстро писать.
- **Минусы**: Очень ограниченный выбор. Вы никогда не попадете точно в фирменный цвет бренда, используя только названия.

---

## 🔢 2. Шестнадцатеричный код (HEX)

Это стандарт де-факто в веб-дизайне. Цвет записывается в виде шести символов после решетки `#`.

```css
.header { background-color: #ff5733; }
.text { color: #333333; }
```

### Как это работает?
Код состоит из трех пар: `#RRGGBB` (Red, Green, Blue).
- Каждая пара — это число в 16-ричной системе (от `00` до `ff`).
- `#ff0000` — чистый красный.
- `#00ff00` — чистый зеленый.
- `#0000ff` — чистый синий.

> [!tip] Сокращенная запись
> Если в каждой паре символы одинаковые, код можно сократить до трех знаков.
> `#ff0000` → `#f00`
> `#cccccc` → `#ccc`

## 🧪 3. Формат RGB и RGBA

Эта система описывает, сколько красного, зеленого и синего нужно смешать (как в палитре художника, но со светом).

```css
/* Значения от 0 до 255 */
.box { background-color: rgb(255, 87, 51); }
```

### Прозрачность (RGBA)
Буква `a` (Alpha) добавляет четвертое значение — прозрачность от `0` (полностью прозрачный) до `1` (полностью непрозрачный).

```css
/* Черный цвет с прозрачностью 50% */
.overlay { background-color: rgba(0, 0, 0, 0.5); }
```

---

## 🌈 4. Формат HSL и HSLA

Это самый «человечный» формат. Его обожают дизайнеры, потому что он интуитивно понятен.

**HSL** расшифровывается как:
1. **Hue (Тон)**: Цвет на цветовом круге (от 0 до 360 градусов). 0 — красный, 120 — зеленый, 240 — синий.
2. **Saturation (Насыщенность)**: Насколько цвет сочный (от 0% до 100%). 0% — это серый цвет.
3. **Lightness (Светлота)**: Насколько цвет светлый или темный (от 0% — черный до 100% — белый).

```css
/* Яркий зеленый */
.grass { background-color: hsl(120, 100%, 50%); }

/* Тот же зеленый, но темнее и спокойнее */
.forest { background-color: hsl(120, 50%, 20%); }
```

### Почему HSL — это круто?
Если вы хотите сделать цвет кнопки чуть темнее при наведении, в HEX вам придется подбирать новый код. В HSL вы просто уменьшаете третье число (Lightness) на 10% — и готово!

## 🏁 Сравнение форматов

| Формат | Пример | Когда использовать? |
|---|---|---|
| **Keywords** | `red` | Для быстрых тестов и простых макетов. |
| **HEX** | `#f05a28` | Стандарт для большинства проектов. Удобно копировать из Photoshop/Figma. |
| **RGB** | `rgb(240, 90, 40)` | Если нужно математически высчитать цвет или использовать прозрачность. |
| **HSL** | `hsl(15, 80%, 55%)` | Лучший выбор для создания цветовых схем и управления яркостью. |

## 💡 Полезные инструменты

Вам не нужно учить коды цветов наизусть. Используйте помощников:
1. **Color Picker в VS Code**: Наведите на квадрат с цветом в коде, и появится палитра.
2. **Adobe Color**: Крутой сервис для подбора сочетающихся цветов.
3. **Coolors.co**: Генератор цветовых палитр за один клик.
4. **Пипетка в DevTools**: Нажмите F12 в браузере, найдите любой цвет и кликните на него — откроется инструмент подбора.

## ♿ Доступность и контраст

Выбирая цвет текста и фона, вы должны помнить о людях с нарушениями зрения.
- Текст должен быть четко виден на фоне.
- Существует стандарт **WCAG**, который определяет коэффициент контрастности.
- В инструментах разработчика Chrome/Firefox есть встроенная проверка: если контраст плохой, браузер покажет предупреждающий значок.

> [!warning] Правило безопасности
> Никогда не используйте светло-серый текст на белом фоне. Это выглядит стильно, но его невозможно читать.

## 🧪 Практика: Эксперименты с цветом

Давайте создадим несколько блоков и раскрасим их разными способами.

```html
<div class="palette">
  <div class="color-box hex">HEX</div>
  <div class="color-box rgb">RGB</div>
  <div class="color-box hsl">HSL</div>
  <div class="color-box alpha">RGBA</div>
</div>
```

```css
.color-box {
  width: 150px;
  height: 100px;
  display: inline-block;
  margin: 10px;
  line-height: 100px;
  text-align: center;
  color: white;
  font-weight: bold;
}

.hex { background-color: #6200ee; }
.rgb { background-color: rgb(3, 218, 198); }
.hsl { background-color: hsl(280, 100%, 60%); }

/* Полупрозрачный синий поверх серого фона страницы */
.alpha {
  background-color: rgba(98, 0, 238, 0.3);
  color: black;
}
```

## ⚠️ Типичные ошибки

1. **Забытая решетка**: Написали `ff5733` вместо `#ff5733`. Стиль не сработает.
2. **Путаница в RGB**: Написали `rgb(300, 0, 0)`. Максимальное значение — 255. Браузер просто приравняет 300 к 255.
3. **Процент в RGB**: В формате `rgb()` нельзя использовать проценты (только в `hsl()`). Хотя в новых стандартах это меняется, лучше придерживаться классики.
4. **Плохой контраст**: Выбор цветов, которые «сливаются» друг с другом.

## ❓ Часто задаваемые вопросы

### «Какой формат цвета самый быстрый для браузера?»
Разница настолько ничтожна, что об этом не стоит беспокоиться. Используйте тот формат, который удобен вам и вашей команде.

### «Что такое currentColor?»
Это специальное ключевое слово в CSS. Оно означает «использовать тот же цвет, который сейчас у текста». Очень удобно для окрашивания рамок (`border`) или иконок.

### «Как сделать градиент?»
Градиенты — это не цвет, а изображение (`background-image`). Мы изучим их чуть позже в уроке про фоны.

### «Видят ли все люди цвета одинаково?»
Нет. Около 8% мужчин страдают дальтонизмом. При создании интерфейсов старайтесь дублировать информацию не только цветом, но и иконками или текстом (например, красная рамка у поля + надпись «Ошибка»).

## 🔑 Ключевые мысли урока

> [!abstract] Запомни
> 1. **HEX (`#RRGGBB`)** — самый популярный и компактный формат.
> 2. **RGB** — управление через красный, зеленый и синий (0-255).
> 3. **HSL** — самый удобный для человека (Тон, Насыщенность, Светлота).
> 4. **Alpha-канал (`rgba`, `hsla`)** позволяет управлять прозрачностью.
> 5. Всегда следите за **контрастностью** текста и фона.
> 6. Современные инструменты (VS Code, DevTools) позволяют легко переключаться между форматами.
