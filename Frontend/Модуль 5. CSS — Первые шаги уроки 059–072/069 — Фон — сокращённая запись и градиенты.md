## 📖 Введение: Магия в одной строке

Мы изучили пять разных свойств для управления фоном: цвет, путь к файлу, повторение, позицию и размер. Писать пять строк кода для каждого блока — долго и неудобно. К счастью, CSS позволяет объединить их все в одно супер-свойство — **`background`**.

Кроме того, мы разберем одну из самых красивых возможностей современного CSS — **градиенты**. Это плавные переходы между цветами, которые браузер рисует сам. Они не весят ни одного килобайта и выглядят потрясающе на любых экранах.

---

## 🏎️ 1. Сокращенная запись: `background`

Вместо того чтобы писать так:
```css
.box {
  background-color: #333;
  background-image: url('pattern.png');
  background-repeat: no-repeat;
  background-position: center;
  background-size: cover;
  background-attachment: fixed;
}
```

Мы можем написать всё в одну строку:
```css
.box {
  background: #333 url('pattern.png') no-repeat center / cover fixed;
}
```

### ⚠️ Правила записи (Важно!):
1. **Порядок**: Почти любые значения можно менять местами, НО...
2. **Размер**: Размер (`background-size`) может быть указан **только после** позиции через косую черту `/`. Например: `center / cover`.
3. **Цвет**: Если вы указываете цвет, он всегда должен быть последним в списке (но браузеры прощают, если он первый).
4. **Необязательность**: Вы можете пропустить любое значение, и браузер установит его по умолчанию.

---

## 🌈 2. Линейные градиенты: `linear-gradient`

Градиент в CSS считается «фоновым изображением» (`background-image`), потому что у него нет одного фиксированного цвета.

### Базовый синтаксис:
```css
.gradient-box {
  /* От верхнего края (по умолчанию) */
  background: linear-gradient(blue, green);
}
```

### Направление:
Вы можете указать, куда движется цвет:
- Ключевые слова: `to right`, `to top left`, `to bottom`.
- Углы: `45deg`, `180deg`, `90deg`.

```css
.diagonal {
  background: linear-gradient(45deg, red, yellow);
}
```

### Цветовые стопы:
Вы можете использовать сколько угодно цветов и указывать, где именно один должен переходить в другой (в процентах).
```css
.rainbow {
  background: linear-gradient(to right, red 0%, yellow 50%, green 100%);
}
```

---

## 🎯 3. Радиальные градиенты: `radial-gradient`

Цвета расходятся из центра (или любой другой точки) кругами или эллипсами.

```css
.circle {
  /* Цвет из центра (белый) к краям (синий) */
  background: radial-gradient(white, blue);
}
```

Вы также можете указать форму (`circle` или `ellipse`) и позицию центра:
```css
.sun {
  background: radial-gradient(circle at top right, yellow, orange);
}
```

---

## 📸 4. Наложение градиента на фото (Overlay)

Это секретный прием всех крутых дизайнеров. Если вы положите белый текст на светлую картинку, он не будет читаться. Если вы наложите на картинку полупрозрачный градиент — текст станет четким.

Помните, мы говорили, что можно указывать несколько фонов?
```css
.hero {
  background:
    linear-gradient(rgba(0,0,0,0.7), rgba(0,0,0,0.3)), /* Верхний слой (черный прозрачный) */
    url('mountain.jpg') no-repeat center / cover;     /* Нижний слой (картинка) */
}
```

---

## 💡 Полезные инструменты для градиентов

Писать сложные градиенты вручную — это мучение. Пользуйтесь онлайн-генераторами:
1. **CSS Gradient (cssgradient.io)**: Визуальный редактор, где можно просто двигать ползунки и копировать готовый код.
2. **Grabient**: Коллекция очень красивых готовых градиентов.
3. **UI Gradients**: Тысячи сочетаний цветов, подобранных профессиональными дизайнерами.

---

## 🚀 Производительность и поддержка

- **Вес**: Градиенты — это код. Они не требуют загрузки файлов, поэтому сайт открывается быстрее.
- **Поддержка**: Все современные браузеры отлично понимают градиенты. Для очень старых браузеров (типа IE9) всегда указывайте обычный `background-color` перед градиентом как «запасной парашют».

```css
.box {
  background-color: blue; /* Для старых браузеров */
  background: linear-gradient(blue, darkblue); /* Для современных */
}
```

## 🧪 Практика: Создаем эффектную кнопку

Давайте сделаем кнопку, которая выглядит объемной благодаря градиенту.

```html
<button class="glass-button">Нажми меня</button>
```

```css
.glass-button {
  padding: 15px 30px;
  border: none;
  border-radius: 50px;
  color: white;
  font-weight: bold;
  font-family: sans-serif;
  cursor: pointer;

  /* Используем сокращенную запись и градиент */
  background: linear-gradient(135deg, #6e8efb 0%, #a777e3 100%);

  /* Добавим небольшую тень для объема */
  box-shadow: 0 4px 15px rgba(110, 142, 251, 0.4);

  /* Эффект при наведении */
  transition: 0.3s;
}

.glass-button:hover {
  /* Просто инвертируем или немного меняем градиент */
  background: linear-gradient(135deg, #a777e3 0%, #6e8efb 100%);
  transform: translateY(-2px);
}
```

## ⚠️ Типичные ошибки

1. **Неверный порядок в `background`**: Попытка написать `cover / center`. Запомните: сначала «где» (позиция), потом «сколько» (размер).
2. **Забытые запятые в градиенте**: Между цветами и направлением всегда должны быть запятые.
3. **Использование background-color для градиента**: Написать `background-color: linear-gradient(...)`. Градиент — это картинка, он работает только с `background` или `background-image`.
4. **Слишком резкие переходы**: Если вы не укажете проценты или укажете их слишком близко (например, `red 50%, blue 50%`), вы получите четкую линию вместо плавного перехода (хотя иногда это тоже дизайнерский ход).

## ❓ Часто задаваемые вопросы

### «Можно ли сделать градиентную рамку?»
Да, но это сложнее. Обычно для этого используют свойство `border-image` или хитрую манипуляцию с двумя вложенными блоками.

### «Что такое повторяющийся градиент?»
Есть функции `repeating-linear-gradient` и `repeating-radial-gradient`. С их помощью можно легко нарисовать «зебру», тетрадный лист в клетку или полосатый леденец всего парой строк кода.

### «Как сделать текст градиентным?»
Как мы узнали в прошлом уроке: используем градиент на фон блока, а затем `background-clip: text` и `color: transparent`.

## 🔑 Ключевые мысли урока

> [!abstract] Запомни
> 1. **`background`** — универсальный способ написать всё в одну строку.
> 2. В сокращенной записи пишем **позицию / размер** (обязательно через слэш).
> 3. **Градиенты** — это программно нарисованные фоновые изображения.
