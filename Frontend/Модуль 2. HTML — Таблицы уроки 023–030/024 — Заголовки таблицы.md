## 🏷️ Тег `<th>` — заголовок ячейки

В большинстве таблиц первая строка или первый столбец содержат названия категорий. Для таких ячеек вместо обычного `<td>` используется тег **`<th>`** (Table Header).

### В чем разница?

| Характеристика | `<td>` (Data) | `<th>` (Header) |
|---|---|---|
| **Смысл** | Обычные данные | Заголовок (столбца или строки) |
| **Вид (по умолчанию)** | Обычный текст, выравнивание влево | **Жирный текст**, выравнивание по центру |
| **Для роботов** | Просто текст | Ключевая информация для экранных читалок (SEO) |

### Пример использования:

```html
<table border="1">
  <tr>
    <th>Продукт</th> <!-- Заголовок -->
    <th>Количество</th> <!-- Заголовок -->
  </tr>
  <tr>
    <td>Бананы</td>
    <td>5 шт</td>
  </tr>
</table>
```

---

## 🔝 Тег `<caption>` — название таблицы

Чтобы подписать таблицу (дать ей заголовок, который будет отображаться прямо над ней), используется тег **`<caption>`**.

### Правила использования:
1. Пишется **сразу после** открывающего тега `<table>`.
2. В таблице может быть только один `<caption>`.
3. По умолчанию отображается по центру над таблицей.

```html
<table border="1">
  <caption>Список покупок на неделю</caption>
  <tr>
    <th>Товар</th>
    <th>Цена</th>
  </tr>
  <tr>
    <td>Хлеб</td>
    <td>40 руб</td>
  </tr>
</table>
```

---

## 🏗️ Вертикальные заголовки

Заголовки `<th>` можно использовать не только в первой строке, но и в начале каждой строки, если это логично для данных.

```html
<table border="1">
  <tr>
    <th>Имя</th>
    <td>Алексей</td>
  </tr>
  <tr>
    <th>Город</th>
    <td>Москва</td>
  </tr>
</table>
```

---

## 🧪 Практика

Создай таблицу характеристик смартфона. Используй `<caption>`, `<th>` и `<td>`.

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Заголовки таблиц</title>
</head>
<body>
    <table border="1">
        <caption>Характеристики iPhone 15</caption>
        <tr>
            <th>Параметр</th>
            <th>Значение</th>
        </tr>
        <tr>
            <th>Экран</th>
            <td>6.1 дюйма</td>
        </tr>
        <tr>
            <th>Процессор</th>
            <td>A16 Bionic</td>
        </tr>
        <tr>
            <th>Память</th>
            <td>128/256/512 ГБ</td>
        </tr>
    </table>
</body>
</html>
```

## 🔑 Ключевые мысли урока

> [!abstract] Запомни
> 1. **`<th>`** — для заголовков ячеек (жирный текст по умолчанию).
> 2. **`<caption>`** — для названия всей таблицы.
> 3. Использование `<th>` делает таблицу понятнее для людей с нарушениями зрения (экранные читалки понимают, к какому заголовку относятся данные).
