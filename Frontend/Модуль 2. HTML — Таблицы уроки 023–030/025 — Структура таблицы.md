## 🏗️ Семантическая структура

Когда таблица становится большой, важно разделить её на логические блоки. Для этого в HTML существуют специальные «контейнеры», которые оборачивают группы строк `<tr>`.

### Три главных блока:

1.  **`<thead>`** (Table Head) — «голова» таблицы. Здесь обычно находятся заголовки столбцов.
2.  **`<tbody>`** (Table Body) — «тело» таблицы. Основные данные.
3.  **`<tfoot>`** (Table Foot) — «футер» или «подвал». Итоги, суммы, примечания.

---

## 📝 Синтаксис

Эти теги пишутся **внутри** `<table>`, но **снаружи** `<tr>`.

```html
<table>
  <caption>Отчет по продажам</caption>

  <thead>
    <tr>
      <th>Товар</th>
      <th>Цена</th>
      <th>Кол-во</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>Ноутбук</td>
      <td>50 000</td>
      <td>2</td>
    </tr>
    <tr>
      <td>Мышь</td>
      <td>1 500</td>
      <td>10</td>
    </tr>
  </tbody>

  <tfoot>
    <tr>
      <td>Итого:</td>
      <td>115 000</td>
      <td>12</td>
    </tr>
  </tfoot>
</table>
```

---

## 🤔 Зачем это нужно?

На первый взгляд, в браузере ничего не меняется. Но эти теги решают важные задачи:

| Причина | Описание |
|---|---|
| **Семантика** | Браузер и поисковики четко понимают, где заголовки, а где данные. |
| **Печать** | Если таблица длинная и не влезает на один лист, умные браузеры при печати будут повторять `<thead>` на каждой новой странице. |
| **Прокрутка** | Можно через CSS зафиксировать `<thead>`, чтобы при прокрутке длинной таблицы заголовки всегда были сверху. |
| **Стилизация** | Легко покрасить все строки в «теле» (tbody) разом, не затрагивая шапку и подвал. |

> [!tip] Порядок в коде
> Даже если в таблице несколько блоков `<tbody>`, `<thead>` всегда должен быть один (в начале), а `<tfoot>` — один (в конце).

## 🧪 Практика

Давай создадим таблицу успеваемости студента.

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Структура таблицы</title>
</head>
<body>
    <table border="1">
        <caption>Успеваемость за семестр</caption>
        <thead>
            <tr>
                <th>Предмет</th>
                <th>Оценка</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>Математика</td>
                <td>5</td>
            </tr>
            <tr>
                <td>История</td>
                <td>4</td>
            </tr>
            <tr>
                <td>Информатика</td>
                <td>5</td>
            </tr>
        </tbody>
        <tfoot>
            <tr>
                <td>Средний балл:</td>
                <td>4.6</td>
            </tr>
        </tfoot>
    </table>
</body>
</html>
```

> [!abstract] Запомни
> 1. **`<thead>`**, **`<tbody>`**, **`<tfoot>`** — это контейнеры для строк.

> 3. Помогают при печати длинных таблиц и их стилизации.
