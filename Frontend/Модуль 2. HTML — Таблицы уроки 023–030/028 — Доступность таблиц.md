## 👂 Как работают экранные читалки?

Люди с нарушениями зрения используют программы (скринридеры), которые читают текст с экрана. Когда такая программа доходит до таблицы, она не видит сетку целиком. Она читает ячейку за ячейкой.

Если таблица размечена плохо, пользователь услышит просто набор цифр и слов. Если хорошо — программа скажет: *«Столбец: Цена. Строка: Хлеб. Значение: 40 рублей»*.

---

## 🎯 Атрибут `scope`

Это самый простой и эффективный способ улучшить доступность. Он явно говорит браузеру, к чему относится заголовок `<th>`.

**Значения:**
- `scope="col"` — заголовок для **столбца**.
- `scope="row"` — заголовок для **строки**.

### Пример:

```html
<table>
  <thead>
    <tr>
      <th scope="col">Сотрудник</th>
      <th scope="col">Должность</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">Александр</th> <!-- Имя — это заголовок строки -->
      <td>Ведущий разработчик</td>
    </tr>
  </tbody>
</table>
```

---

## 🏷️ Дополнительное описание

### `<caption>` — ваш лучший друг
Мы уже учили этот тег. Он критически важен для доступности, так как это первое, что прочитает скринридер.

### `aria-describedby`
Если у таблицы есть сложное описание (например, абзац текста под ней), можно связать их.

```html
<p id="table-desc">Данная таблица показывает рост прибыли за последние 5 лет в миллионах долларов.</p>

<table aria-describedby="table-desc">
  <!-- содержимое -->
</table>
```

## ✅ Чек-лист доступной таблицы

1.  Используются `<th>` для всех заголовков.
2.  Добавлен `<caption>` с кратким названием.
3.  Для всех `<th>` прописан атрибут `scope`.
4.  Соблюдена семантика (`thead`, `tbody`).
5.  **Нет пустых ячеек** для красоты (пустая ячейка может запутать читалку).

## 🧪 Практика

Давай создадим максимально доступную таблицу.

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Доступность таблиц</title>
</head>
<body>

    <table border="1">
        <caption>График работы врачей на текущую неделю</caption>
        <thead>
            <tr>
                <th scope="col">Специализация</th>
                <th scope="col">ФИО врача</th>
                <th scope="col">Кабинет</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <th scope="row">Терапевт</th>
                <td>Иванов И.И.</td>
                <td>102</td>
            </tr>
            <tr>
                <th scope="row">Хирург</th>
                <td>Петров П.П.</td>
                <td>205</td>
            </tr>
            <tr>
                <th scope="row">Офтальмолог</th>
                <td>Сидорова С.С.</td>
                <td>301</td>
            </tr>
        </tbody>
    </table>

</body>
</html>
```
