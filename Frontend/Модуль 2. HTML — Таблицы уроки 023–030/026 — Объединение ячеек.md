- Узнать, как не сломать таблицу при объединении

---

## 🏗️ Что такое объединение?

Иногда данные не умещаются в одну ячейку или относятся сразу к нескольким столбцам/строкам. Для этого используются атрибуты **`colspan`** и **`rowspan`**, которые добавляются к тегам `<td>` или `<th>`.

---

## ↔️ Объединение по горизонтали (`colspan`)

Атрибут **`colspan`** (column span) говорит ячейке занять место нескольких столбцов **вправо**.

**Синтаксис:** `<td colspan="число_столбцов">`

### Пример:
Допустим, нам нужен общий заголовок для двух столбцов:

```html
<table border="1">
  <tr>
    <th colspan="2">Контактная информация</th>
  </tr>
  <tr>
    <td>Телефон</td>
    <td>+7 900...</td>
  </tr>
</table>
```

> [!warning] Важно!
> Если ячейка "съела" место соседа (`colspan="2"`), то в этой строке нужно написать **на один тег `<td>` меньше**, иначе таблица "распухнет".

---

## ↕️ Объединение по вертикали (`rowspan`)

Атрибут **`rowspan`** (row span) заставляет ячейку растянуться на несколько строк **вниз**.

**Синтаксис:** `<td rowspan="число_строк">`

### Пример:
Один и тот же человек работает в разные смены:

```html
<table border="1">
  <tr>
    <th>Смена</th>
    <th>Сотрудник</th>
  </tr>
  <tr>
    <td rowspan="2">Утро</td> <!-- Занимает эту строку и следующую -->
    <td>Иван</td>
  </tr>
  <tr>
    <!-- Здесь ячейка для "Утро" уже не нужна, она пришла сверху -->
    <td>Мария</td>
  </tr>
</table>
```

---

## 🧩 Сложное объединение

Можно использовать оба атрибута одновременно, чтобы создать большую ячейку-блок.

```html
<td colspan="2" rowspan="2">Большой блок</td>
```

---

## 🧪 Практика

Давай создадим сложную структуру. Повтори этот код:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Объединение ячеек</title>
</head>
<body>
    <table border="1">
        <caption>Расписание дежурств</caption>
        <thead>
            <tr>
                <th>День</th>
                <th>8:00 - 12:00</th>
                <th>12:00 - 16:00</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>Понедельник</td>
                <td>Анна</td>
                <td>Дмитрий</td>
            </tr>
            <tr>
                <td>Вторник</td>
                <td colspan="2">Технический перерыв (никто не дежурит)</td>
            </tr>
            <tr>
                <td>Среда</td>
                <td rowspan="2">Сергей (двойная смена)</td>
                <td>Елена</td>
            </tr>
            <tr>
                <td>Четверг</td>
                <!-- Тут только одна ячейка, так как первая занята Сергеем -->
                <td>Олег</td>
            </tr>
        </tbody>
    </table>
</body>
</html>
```

## 🔑 Ключевые мысли урока

> [!abstract] Запомни
> 1. **`colspan`** — растягивает ячейку вширь (вправо).
> 2. **`rowspan`** — растягивает ячейку вглубь (вниз).
> 3. Если ячейка объединена, лишние `<td>` в строке/столбце нужно **удалять**.
> 4. Значение атрибута — это количество ячеек, которые должны стать одной (минимум "2").
