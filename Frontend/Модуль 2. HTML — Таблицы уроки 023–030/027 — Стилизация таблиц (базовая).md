## 🎨 Почему CSS, а не атрибуты?

В старых уроках можно встретить атрибуты типа `border="1"`, `cellpadding="10"` или `bgcolor="red"`. **Забудь про них.** В современном вебе всё оформление делается через CSS. Это позволяет менять дизайн всех таблиц на сайте разом.

---

## 🏗️ Свойство `border-collapse`

По умолчанию браузер рисует границы ячеек отдельно друг от друга (между ними есть зазор). Чтобы таблица выглядела привычно, границы нужно «схлопнуть».

```css
table {
  border-collapse: collapse; /* Границы ячеек сливаются в одну */
}
```

| Значение | Эффект |
|---|---|
| **`separate`** | Границы разделены (по умолчанию). |
| **`collapse`** | Общие границы (рекомендуется). |

---

## 📏 Размеры и отступы

Для таблиц отлично работают стандартные свойства:
- **`width`** — ширина таблицы (например, `100%` для всей ширины экрана).
- **`padding`** — отступы внутри ячеек (чтобы текст не лип к границам).

```css
th, td {
  border: 1px solid black; /* Рамка толщиной 1px */
  padding: 10px;           /* Отступ от рамки до текста */
  text-align: left;        /* Выравнивание текста */
}
```

---

## 🦓 Полосатая таблица (Zebra Stripes)

Длинные таблицы сложно читать. Чтобы глаз не терял строку, их делают полосатыми с помощью псевдокласса `:nth-child(even)` (четные) или `:nth-child(odd)` (нечетные).

```css
/* Покрасим каждую четную строку в светло-серый цвет */
tbody tr:nth-child(even) {
  background-color: #f2f2f2;
}

/* Изменим цвет при наведении курсора */
tr:hover {
  background-color: #e1f5fe;
}
```

## 🧪 Практика

Давай создадим полноценную стилизованную таблицу.

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Базовая стилизация</title>
    <style>
        table {
            width: 100%;
            border-collapse: collapse;
            font-family: Arial, sans-serif;
            margin-bottom: 20px;
        }

        caption {
            font-size: 1.5em;
            margin: 10px;
            font-weight: bold;
        }

        th, td {
            border: 1px solid #ddd;
            padding: 12px;
            text-align: left;
        }

        thead th {
            background-color: #4CAF50;
            color: white;
        }

        tbody tr:nth-child(even) {
            background-color: #f9f9f9;
        }

        tbody tr:hover {
            background-color: #f1f1f1;
            cursor: pointer;
        }
    </style>
</head>
<body>

    <table>
        <caption>Список сотрудников</caption>
        <thead>
            <tr>
                <th>Имя</th>
                <th>Должность</th>
                <th>Отдел</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>Иван Иванов</td>
                <td>Дизайнер</td>
                <td>Креатив</td>
            </tr>
            <tr>
                <td>Мария Петрова</td>
                <td>Разработчик</td>
                <td>IT</td>
            </tr>
            <tr>
                <td>Петр Сидоров</td>
                <td>Менеджер</td>
                <td>Продажи</td>
            </tr>
            <tr>
                <td>Анна Кузнецова</td>
                <td>HR</td>
                <td>Кадры</td>
            </tr>
        </tbody>
    </table>

</body>
</html>
```

## 🔑 Ключевые мысли урока

> [!abstract] Запомни
> 1. **`border-collapse: collapse`** — маст-хэв для любой нормальной таблицы.
> 2. Стилизуй `th` и `td` отдельно, чтобы выделить заголовки.
> 3. **`padding`** — лучший способ сделать таблицу читаемой.
> 4. Полосатость и эффекты при наведении (`:hover`) помогают пользователю ориентироваться в данных.
