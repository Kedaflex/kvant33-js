## 📖 Введение: Порядок имеет значение

В прошлом уроке мы выбирали элементы по их «поведению» (навели мышку — покрасили). Сегодня мы будем выбирать их по их «месту в строю».

Представьте, что у вас есть список из 100 товаров. Вы хотите, чтобы у каждого второго был серый фон (эффект «зебры»), чтобы пользователю было легче читать. Добавлять класс `.gray` каждому второму товару вручную — это безумие.

Или вы хотите, чтобы у первого элемента в меню не было отступа слева, а у последнего — справа.

Для таких задач в CSS есть **структурные псевдоклассы**. Они позволяют выбирать элементы, основываясь на их порядковом номере внутри родителя.

---

## 👶 Простые родственники

### 1. :first-child (Первенец)
Выбирает элемент, только если он является **самым первым** ребенком своего родителя.
```css
/* Убрать верхний отступ у первого абзаца в статье */
p:first-child {
  margin-top: 0;
}
```

### 2. :last-child (Последний в роду)
Выбирает элемент, если он является **последним** ребенком.
```css
/* Убрать разделительную линию у последнего пункта меню */
li:last-child {
  border-bottom: none;
}
```

### 3. :only-child (Единственный ребенок)
Срабатывает, если у родителя нет других детей. Идеально для стилизации «одиноких» сообщений или уведомлений.

---

## 🧬 Магия :nth-child(n)

Это самый мощный структурный селектор. Буква `n` в скобках — это формула или число.

### Вариант 1: Просто число
Выбирает конкретный элемент по номеру.
- `li:nth-child(3)` — выберет только третий элемент.

### Вариант 2: Ключевые слова (Зебра)
- `li:nth-child(odd)` — выберет все **нечетные** элементы (1, 3, 5...).
- `li:nth-child(even)` — выберет все **четные** элементы (2, 4, 6...).

### Вариант 3: Формула (an + b)
Это позволяет создавать циклы любой сложности.
- `li:nth-child(3n)` — каждый третий (3, 6, 9...).
- `li:nth-child(3n+1)` — каждый третий, начиная с первого (1, 4, 7...).
- `li:nth-child(n+5)` — все элементы, начиная с пятого и до конца.
- `li:nth-child(-n+3)` — только первые три элемента.

---

## 🔍 Отличие -child от -of-type

Это тонкий момент, на котором ловят новичков.
- `p:nth-child(2)` — ищет любой второй элемент. Если вторым ребенком окажется `div`, а не `p`, то селектор **не сработает**.
- `p:nth-of-type(2)` — ищет именно второй по счету **абзац**, игнорируя другие теги (`div`, `img`), которые могут стоять между ними.

> [!tip] Совет
> Если в вашем контейнере вперемешку лежат разные теги, всегда используйте `-of-type`. Если только однотипные (например, `li` внутри `ul`) — достаточно `-child`.

---

## 🎨 Практические примеры

### Эффект «Зебры» для таблицы
```css
tr:nth-child(even) {
  background-color: #f2f2f2;
}
```

### Первая буква (через псевдокласс)
Хотя для этого есть псевдоэлемент, иногда используют `:first-of-type` для стилизации первого абзаца статьи:
```css
article p:first-of-type {
  font-size: 1.2em;
  font-weight: bold;
}
```

### Сетка (Grid/Flex)
Если у вас в ряду по 3 карточки, и вы хотите добавить отступ каждой правой:
```css
.card:nth-child(3n) {
  margin-right: 0;
}
```

## ❓ Часто задаваемые вопросы

### «С какого числа начинается отсчет?»
В отличие от программирования, где всё начинается с 0, в CSS отсчет человеческий: **с единицы (1)**. Первый элемент — это 1.

### «Почему `:last-child` не работает?»
Чаще всего это происходит потому, что в конце родителя затаился какой-то незаметный тег (например, `<script>` или скрытый `div`), который технически является последним ребенком, но не соответствует вашему селектору.

### «Можно ли комбинировать псевдоклассы?»
Да! `li:nth-child(even):hover` — выберет четный элемент, на который навели мышку.

## 🧪 Практика: Умный список дел

Создадим список, который сам расставляет акценты.

```html
<!DOCTYPE html>
<html lang="ru">
<head>
  <style>
    ul {
      list-style: none;
      padding: 0;
      max-width: 300px;
      border: 1px solid #ddd;
    }

    li {
      padding: 15px;
      border-bottom: 1px solid #eee;
    }

    /* Первый элемент - заголовок */
    li:first-child {
      background-color: #3498db;
      color: white;
      font-weight: bold;
    }

    /* Последний элемент без нижней границы */
    li:last-child {
      border-bottom: none;
    }

    /* Чередование фона для списка */
    /* Начинаем со второго, чтобы не трогать заголовок */
    li:nth-child(even) {
      background-color: #f9f9f9;
    }

    /* Выделим третий элемент специально */
    li:nth-child(3) {
      color: #e74c3c;
    }

    /* Каждому второму добавим иконку в конце (через псевдоэлемент, который изучим далее) */
    li:nth-child(2n)::after {
      content: " ⭐";
    }
  </style>
</head>
<body>

  <ul>
    <li>Список задач</li>
    <li>Купить молоко</li>
    <li>Позвонить маме</li>
    <li>Сделать домашку по CSS</li>
    <li>Покормить кота</li>
    <li>Выспаться</li>
  </ul>

</body>
</html>
```

## 🔑 Ключевые мысли урока

> [!abstract] Запомни
> 1. Структурные псевдоклассы выбирают элементы по их **положению** в коде.
> 2. Отсчет начинается с **1**.
> 3. `:nth-child(even/odd)` — для чередования (зебры).
> 4. Формула `an+b` позволяет выбрать любой цикл.
> 5. `-of-type` безопаснее, если в контейнере разные типы тегов.
> 6. Эти селекторы избавляют от необходимости плодить лишние классы в HTML.

## 🏁 Итог

Мы научились выбирать элементы по их «порядковому номеру». Это делает наш CSS по-настоящему системным. Теперь мы можем управлять списками и таблицами любой длины.

Но псевдоклассы выбирают **целые** элементы. А что если мы хотим выбрать только **часть** элемента? Например, только первую букву абзаца? Или мы хотим вставить декоративную иконку перед текстом, не добавляя её в HTML?
