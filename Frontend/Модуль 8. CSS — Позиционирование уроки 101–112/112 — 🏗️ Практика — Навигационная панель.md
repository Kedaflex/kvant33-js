# 🏗️ Практика: Современная навигационная панель

> - Использовать псевдоэлементы для декоративных эффектов.

---

## 📖 Введение: Лицо вашего сайта

Навигационная панель (шапка или Header) — это самый важный элемент интерфейса. Она должна быть доступной, понятной и всегда находиться под рукой.

Сегодня мы создадим профессиональную шапку, которая:
1. **Фиксируется** сверху при прокрутке.
2. Содержит **выпадающий список** услуг.
3. Имеет красивые **эффекты наведения**, созданные с помощью псевдоэлементов.

Это задание объединит всё, что мы выучили в этом модуле. Мы будем использовать «современный минимализм»: много воздуха, тонкие линии и плавные анимации.

---

## 📐 План работы

1. **HTML-разметка**: Используем семантический тег `<nav>` и вложенные списки `<ul>`.
2. **Основа**: Фиксируем шапку и настраиваем её размеры.
3. **Меню**: Выстраиваем пункты в ряд (на блочной модели) и стилизуем ссылки.
4. **Выпадающий список**: Самая сложная часть — прячем меню и показываем его только при наведении с помощью `position: absolute`.
5. **Декор**: Добавляем подчеркивания и микро-анимации.

## 🛠️ Шаг 1: HTML-разметка

```html
<header class="header">
  <div class="container">
    <a href="#" class="logo">Minimal<span>Studio</span></a>

    <nav class="nav">
      <ul class="nav-list">

        <li class="has-dropdown">
          <a href="#" class="nav-link">Услуги</a>
          <ul class="dropdown">
            <li><a href="#">Веб-дизайн</a></li>
            <li><a href="#">Разработка</a></li>
            <li><a href="#">Маркетинг</a></li>
          </ul>
        </li>
        <li><a href="#" class="nav-link">Портфолио</a></li>
        <li><a href="#" class="nav-link">Контакты</a></li>
      </ul>
    </nav>
  </div>
</header>

<main class="content">
  <!-- Много текста для проверки прокрутки -->
</main>
```

## 🎨 Шаг 2: Базовые стили и Фиксация

```css
*, *::before, *::after {
  box-sizing: border-box;
}

body {
  margin: 0;
  font-family: 'Inter', sans-serif;
  background-color: #fcfcfc;
}

.header {
  position: fixed; /* Фиксируем сверху */
  top: 0;
  left: 0;
  width: 100%;
  height: 80px;
  background-color: rgba(255, 255, 255, 0.95);
  backdrop-filter: blur(10px); /* Эффект матового стекла */
  border-bottom: 1px solid #eee;
  z-index: 1000; /* Поверх всего */
}

.container {
  max-width: 1200px;
  margin: 0 auto;
  height: 100%;
  padding: 0 20px;
  display: flex; /* Маленький спойлер будущего: центрируем элементы в ряд */
  align-items: center;
  justify-content: space-between;
}

/* Компенсируем высоту фиксированной шапки */
.content {
  padding-top: 100px;
  height: 2000px; /* Чтобы был скролл */
}
```

## 🔗 Шаг 3: Навигация и Ссылки

Используем `inline-block` и `relative` для декора.

```css
.nav-list {
  list-style: none;
  margin: 0;
  padding: 0;
}

.nav-list > li {
  display: inline-block;
  margin-left: 30px;
  position: relative; /* Якорь для выпадающего меню */
}

.nav-link {
  text-decoration: none;
  color: #333;
  font-weight: 500;
  padding: 10px 0;
  display: block;
  transition: color 0.3s;
}

/* Красивое подчеркивание через ::after */
.nav-link::after {
  content: "";
  position: absolute;
  bottom: 0;
  left: 0;
  width: 0;
  height: 2px;
  background-color: #000;
  transition: width 0.3s ease;
}

.nav-link:hover::after {
  width: 100%;
}
```

## 📂 Шаг 4: Выпадающее меню (Магия Absolute)

```css
.dropdown {
  position: absolute;
  top: 100%; /* Прямо под пунктом меню */
  left: 0;
  background-color: white;
  min-width: 200px;
  padding: 15px 0;
  list-style: none;
  border: 1px solid #eee;
  border-radius: 12px;
  box-shadow: 0 10px 30px rgba(0,0,0,0.1);

  /* Прячем меню по умолчанию */
  opacity: 0;
  visibility: hidden;
  transform: translateY(10px);
  transition: all 0.3s ease;
}

/* Показываем при наведении на родителя */
.has-dropdown:hover .dropdown {
  opacity: 1;
  visibility: visible;
  transform: translateY(0);
}

.dropdown a {
  display: block;
  padding: 10px 20px;
  text-decoration: none;
  color: #666;
  font-size: 14px;
  transition: background 0.2s, color 0.2s;
}

.dropdown a:hover {
  background-color: #f9f9f9;
  color: #000;
}
```

## 🔍 Разбор ключевых решений

1. **position: fixed**: Мы прижали шапку к верху. Чтобы она не была «прозрачной» для кликов, мы задали ей `width: 100%` и `z-index: 1000`.
2. **position: relative** (на пункте меню): Это позволило нам привязать выпадающий список (`dropdown`) именно к слову «Услуги», а не к краю всего экрана.
3. **opacity + visibility**: Мы не использовали `display: none`, потому что его нельзя плавно анимировать. Комбинация `opacity` и `visibility` позволяет сделать меню невидимым для глаз и для мышки, но при этом плавно «проявить» его при наведении.
4. **backdrop-filter**: Мы добавили современный эффект размытия фона под шапкой, что придает интерфейсу глубину.

## 🧪 Практика: Интерактивы и отладка

### Что может пойти не так?
- **Разрыв при наведении**: Если между пунктом меню и выпадающим списком есть пустое пространство (хотя бы 1 пиксель), то при попытке перевести мышку на список, состояние `:hover` на родителе пропадет, и список исчезнет.
- **Решение**: Убедитесь, что `top: 100%` или используйте невидимый псевдоэлемент, который заполнит пустоту.

## ❓ Часто задаваемые вопросы

### «Почему вы использовали и `opacity`, и `visibility`?»
Потому что если оставить только `opacity: 0`, меню будет невидимым, но оно всё еще будет «существовать» и перекрывать ссылки под ним. Вы не сможете нажать на текст в основном контенте, если там прячется прозрачное меню. `visibility: hidden` делает его неактивным для мыши.

### «Как сделать так, чтобы меню открывалось по клику, а не по наведению?»
На чистом CSS это сделать сложно (можно использовать «трюк с чекбоксом»). Обычно для открытия меню по клику используют JavaScript. Мы изучим это в блоке JS.

### «Будет ли такая шапка работать на мобильных?»
Не совсем. На мобильных экранах пункты меню не поместятся в ряд. Для них обычно делают «бургер-меню». В следующем модуле мы научимся адаптировать такие элементы.

## 🔑 Ключевые мысли модуля

> [!abstract] Запомни
> 1. **Нормальный поток** — ваш друг. Не вырывайте из него элементы без нужды.
> 2. **Relative** — это якорь, **Absolute** — это полет.
> 3. **Fixed** и **Sticky** управляют поведением при прокрутке.
> 4. **z-index** работает только внутри своего контекста наложения.
> 5. Псевдоклассы и псевдоэлементы позволяют создавать сложную логику без лишнего кода.
> 6. Профессиональная верстка — это комбинация всех этих методов.
