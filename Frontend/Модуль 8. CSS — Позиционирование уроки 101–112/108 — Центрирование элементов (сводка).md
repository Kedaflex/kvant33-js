## 📖 Введение: Главная боль верстальщика

Если вы спросите любого разработчика, какая задача в CSS самая частая и при этом самая раздражающая (особенно в начале пути), ответ будет один: **центрирование**.

Казалось бы, простая задача — поставить блок посередине. Но способы будут кардинально отличаться в зависимости от того, центрируете вы текст внутри строки, картинку внутри блока или целый блок внутри окна браузера.

В этом уроке мы не будем изучать новые свойства, мы соберем воедино все наши инструменты. Это ваша «шпаргалка по центрированию», к которой вы будете возвращаться еще не раз.

---

## 📝 1. Центрирование текста и строчных элементов

Если ваш контент — это текст, ссылки или картинки внутри абзаца, используйте свойство `text-align`.

- **Что делать**: Задать родителю `text-align: center`.
- **Особенности**: Работает только для `inline` и `inline-block` дочерних элементов.

```css
.parent {
  text-align: center;
}
```

---

## 🧱 2. Горизонтальное центрирование блоков

Если вам нужно отцентрировать сам контейнер (например, `div`) внутри другого контейнера.

- **Что делать**: Задать блоку фиксированную ширину (`width` или `max-width`) и автоматические отступы по бокам.
- **Свойство**: `margin: 0 auto;` (или `margin-left: auto; margin-right: auto;`).

```css
.child-block {
  width: 80%;
  max-width: 1000px;
  margin: 0 auto;
}
```

---

## 🚀 3. Центрирование через позиционирование (Классика)

До появления Flexbox это был единственный способ отцентрировать элемент и по горизонтали, и по вертикали одновременно. Этот метод до сих пор часто используется для модальных окон.

### Способ А: Известные размеры
Если вы точно знаете ширину и высоту блока.
```css
.centered-absolute {
  position: absolute;
  top: 50%;
  left: 50%;
  width: 300px;
  height: 200px;
  /* Сдвигаем назад на половину своей ширины и высоты */
  margin-top: -100px;
  margin-left: -150px;
}
```

### Способ Б: Неизвестные размеры (Современный)
Если блок может менять размер в зависимости от текста. Используем `transform` (мы изучим его подробно позже, но запомните этот рецепт сейчас).
```css
.centered-absolute-smart {
  position: absolute;
  top: 50%;
  left: 50%;
  /* Сдвигаем назад на 50% от СОБСТВЕННОГО размера */
  transform: translate(-50%, -50%);
}
```

---

## ⚓ 4. Вертикальное центрирование одной строки

Если у вас есть кнопка или меню с фиксированной высотой, и вы хотите, чтобы текст в ней стоял ровно посередине.

- **Что делать**: Задать свойству `line-height` значение, равное высоте блока (`height`).
- **Особенности**: Работает только для одной строки текста.

```css
.button {
  height: 50px;
  line-height: 50px;
}
```

---

## ⚖️ 5. Табличный метод (Ретро-стиль)

Иногда бывает полезно заставить `div` вести себя как ячейка таблицы. В таблицах вертикальное выравнивание работает «из коробки».

```css
.parent {
  display: table;
  height: 300px;
}
.child {
  display: table-cell;
  vertical-align: middle;
  text-align: center;
}
```

---

## 🔮 Взгляд в будущее: Flexbox

Хотя мы посвятим этому целый модуль, нельзя не упомянуть самый легкий и современный способ. Если вы можете использовать Flexbox — используйте его.

```css
.parent {
  display: flex;
  justify-content: center; /* Центр по горизонтали */
  align-items: center;     /* Центр по вертикали */
}
```

---

## 🔍 Сравнительная таблица методов

| Метод | По горизонтали? | По вертикали? | Требует ширину? | Тип элементов |
| :--- | :---: | :---: | :---: | :--- |
| `text-align: center` | Да | Нет | Нет | Inline / Inline-block |
| `margin: 0 auto` | Да | Нет | **Да** | Block |
| `line-height` | Нет | Да | Нет | Текст (1 строка) |
| `absolute` + `margin` | Да | Да | **Да** | Любой |
| `absolute` + `transform`| Да | Да | Нет | Любой |

## 🧪 Практика: Три способа в одном файле

Посмотрите, как по-разному настраиваются блоки для достижения одной цели.

```html
<!DOCTYPE html>
<html lang="ru">
<head>
  <style>
    .demo { margin-bottom: 50px; border: 1px solid #ccc; position: relative; }

    /* 1. Margin auto */
    .method-1 { height: 100px; }
    .method-1 .child {
      width: 100px; height: 50px; background: red;
      margin: 25px auto;
    }

    /* 2. Text align */
    .method-2 { text-align: center; padding: 20px; }
    .method-2 .child {
      display: inline-block; width: 100px; height: 50px; background: blue;
    }

    /* 3. Absolute */
    .method-3 { height: 200px; }
    .method-3 .child {
      position: absolute;
      top: 50%; left: 50%;
      width: 100px; height: 50px; background: green;
      transform: translate(-50%, -50%);
    }
  </style>
</head>
<body>

  <h3>1. Блочный (margin: auto)</h3>
  <div class="demo method-1"><div class="child"></div></div>

  <h3>2. Строчный (text-align)</h3>
  <div class="demo method-2"><div class="child"></div></div>

  <h3>3. Абсолютный (top: 50% + transform)</h3>
  <div class="demo method-3"><div class="child"></div></div>

</body>
</html>
```

## ❓ Часто задаваемые вопросы

### «Почему `vertical-align: middle` не работает для обычного div?»
Потому что это свойство создавалось для **строчных** элементов (выравнивание относительно базовой линии текста) и **ячеек таблиц**. Обычные блоки в нормальном потоке его просто игнорируют.

### «Как центрировать фоновое изображение?»
Для этого есть отдельное свойство `background-position: center;`. Оно не влияет на HTML-элементы, только на картинку за ними.

### «Что делать, если `margin: 0 auto` не центрирует блок?»
Проверьте три вещи:
1. Задана ли ширина (`width`).
2. Не является ли элемент строчным (`span`, `a`) — им нужно добавить `display: block`.
3. Не применен ли к нему `float` или `position: absolute` — они ломают работу `margin: auto`.
