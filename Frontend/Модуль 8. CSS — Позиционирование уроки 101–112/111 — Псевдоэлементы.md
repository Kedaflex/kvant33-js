## 📖 Введение: Элементы, которых нет в HTML

Представьте, что вам нужно добавить маленькую иконку кавычек перед каждой цитатой. Вы можете зайти в HTML и вручную дописать `<span>"</span>` сто раз. Но это скучно и делает код тяжелым.

А что если бы вы могли приказать CSS: «Создай виртуальный элемент в начале каждой цитаты и напиши там кавычку»?

Именно для этого существуют **псевдоэлементы**. В отличие от псевдоклассов (которые описывают *состояние*), псевдоэлементы позволяют стилизовать конкретную *часть* контента или даже вставить новый контент, которого изначально не было на странице.

В современном стандарте псевдоэлементы пишутся с **двумя двоеточиями** (`::before`), чтобы отличать их от псевдоклассов (`:hover`). Хотя браузеры поймут и одно двоеточие, писать два — признак профессионализма.

## 🏗️ Свойства ::before и ::after

Это два самых часто используемых псевдоэлемента. Они позволяют вставить виртуальный контент в начало или в конец элемента.

### Главное правило: Свойство content
Чтобы псевдоэлемент появился на экране, у него **обязательно** должно быть свойство `content`. Без него (даже если контент — это пустая строка `""`) элемент не будет создан.

```css
.quote::before {
  content: "«"; /* Вставить кавычку в начало */
  color: blue;
  font-size: 2em;
}

.quote::after {
  content: "»"; /* Вставить кавычку в конец */
  color: blue;
}
```

### Как это выглядит в DOM?
Псевдоэлементы вставляются **внутрь** элемента.
- `::before` становится самым первым ребенком.
- `::after` становится самым последним ребенком.

---

## 🎨 Декоративная магия и Позиционирование

Чаще всего `::before` и `::after` используют не для текста, а для графики. С их помощью рисуют линии, кружочки, иконки, тени и блики.

Поскольку это виртуальные элементы, им можно задать `position: absolute`. Это открывает безграничные возможности для дизайна.

```css
.link {
  position: relative; /* Родитель - якорь */
}

/* Рисуем подчеркивание, которое появляется при наведении */
.link::after {
  content: "";
  position: absolute;
  bottom: -5px;
  left: 0;
  width: 0; /* Изначально ширина 0 */
  height: 2px;
  background: red;
  transition: width 0.3s;
}

.link:hover::after {
  width: 100%; /* Растягиваем линию */
}
```

## 🔠 Другие полезные псевдоэлементы

### 1. ::first-letter (Первая буква)
Позволяет создать красивую буквицу (Drop Cap) в начале статьи.
```css
p::first-letter {
  font-size: 3em;
  float: left;
  margin-right: 10px;
  color: darkred;
}
```

### 2. ::first-line (Первая строка)
Стилизует только первую строку текста. Если пользователь изменит размер окна и слова в строке поменяются — стиль всё равно будет применяться только к тому, что попало в первую строку.

### 3. ::selection (Выделение)
Позволяет изменить цвет фона и текста, когда пользователь выделяет его мышкой.
```css
::selection {
  background: yellow;
  color: black;
}
```

## 🔍 Важные ограничения

1. **Не для самозакрывающихся тегов**: Вы не можете применить `::before` или `::after` к тегам `<img>`, `<input>`, `<br>` или `<hr>`. Почему? Потому что эти псевдоэлементы вставляются **внутрь** тега, а у самозакрывающихся тегов нет «внутренностей».
2. **Доступность**: Контент, вставленный через `content: "..."`, не всегда корректно читается программами-экранными дикторами. Поэтому никогда не пишите там важную информацию. Используйте их только для украшений.
3. **Тип отображения**: По умолчанию псевдоэлементы являются **строчными** (`inline`). Если вы хотите задать им `width` или `height`, не забудьте добавить `display: block` или `position: absolute`.

## ❓ Часто задаваемые вопросы

### «Зачем использовать псевдоэлементы, если можно просто добавить картинку в HTML?»
Чтобы не раздувать HTML-код декоративным мусором. Псевдоэлементы позволяют менять дизайн сайта, меняя только CSS. Это гораздо удобнее для поддержки.

### «Можно ли добавить картинку в свойство content?»
Да, можно: `content: url('icon.png');`. Но управлять размером такой картинки будет сложно. Лучше сделать `content: "";` и задать иконку через `background-image`.

### «Как сделать так, чтобы ::before был ПОД основным текстом?»
Используйте `z-index: -1`. Но помните: чтобы это сработало, у основного элемента должен быть создан контекст наложения (например, через `position: relative`).

## 🧪 Практика: Список с кастомными маркерами

Давайте заменим стандартные точки списка на красивые галочки.

```html
<!DOCTYPE html>
<html lang="ru">
<head>
  <style>
    .custom-list {
      list-style: none; /* Убираем стандартные точки */
      padding: 0;
      font-family: sans-serif;
    }

    .custom-list li {
      position: relative;
      padding-left: 35px;
      margin-bottom: 15px;
      line-height: 1.4;
    }

    /* Создаем галочку через псевдоэлемент */
    .custom-list li::before {
      content: "";
      position: absolute;
      left: 0;
      top: 2px;
      width: 20px;
      height: 20px;
      background-color: #2ecc71;
      border-radius: 50%;

      /* Рисуем галочку (уголок) с помощью border */
      border: 2px solid white;
      border-top: none;
      border-right: none;
      transform: rotate(-45deg) scale(0.6);
      /* Смещаем галочку, чтобы она была по центру кружка */
      margin-top: 2px;
    }
  </style>
</head>
<body>

  <ul class="custom-list">
    <li>Выучить основы HTML</li>
    <li>Разобраться с блочной моделью CSS</li>
    <li>Освоить позиционирование</li>
    <li>Научиться использовать псевдоэлементы для декора</li>
  </ul>

</body>
</html>
```

## 🔑 Ключевые мысли урока

> [!abstract] Запомни
> 1. **Псевдоэлементы** создают виртуальные части контента.
> 2. Пишутся с двумя двоеточиями: `::before`, `::after`.
