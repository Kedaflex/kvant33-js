## 📖 Введение: Библиотека мечты

В прошлом уроке мы узнали о «безопасных» шрифтах. Но давайте будем честны: их мало, и они всем надоели. Если вы хотите, чтобы ваш сайт выглядел уникально, стильно и современно, вам нужны **веб-шрифты**.

Веб-шрифты — это файлы шрифтов, которые хранятся на сервере. Когда пользователь заходит на ваш сайт, браузер скачивает эти файлы «на лету» и применяет их к тексту.

Самым крупным, удобным и, что немаловажно, бесплатным хранилищем таких шрифтов является **Google Fonts**. Это огромная библиотека с открытым исходным кодом, где собраны тысячи качественных шрифтов, поддерживающих сотни языков. Сегодня мы научимся «приручать» этого гиганта.

## 🛠️ Как работает Google Fonts?

Принцип работы прост:
1. Вы выбираете шрифт на сайте [fonts.google.com](https://fonts.google.com/).
2. Получаете специальную ссылку (код).
3. Вставляете эту ссылку в свой проект.
4. Используете название шрифта в свойстве `font-family`.

### Основные понятия интерфейса:
- **Glyphs**: Количество доступных символов (буквы, цифры, знаки).
- **Styles**: Количество начертаний (тонкий, жирный, курсив и т.д.).
- **Subsets**: Языковые наборы (например, Latin для английского и Cyrillic для русского).

> [!warning] Важное правило
> Никогда не подключайте все доступные стили шрифта «на всякий случай». Каждый стиль — это отдельный файл, который браузеру нужно скачать. Подключение 10 стилей вместо 2 может замедлить ваш сайт в несколько раз.

## 🔗 Способ №1: Подключение через <link> (Рекомендуемый)

Это самый быстрый и надежный способ. Мы вставляем код в секцию `<head>` нашего HTML-файла **до** того, как подключаем собственный файл стилей.

### Пошаговая инструкция:
1. Выберите нужные начертания (например, Regular 400 и Bold 700).
2. Нажмите "Get embed code" в корзине Google Fonts.
3. Скопируйте блок тегов `<link>`.
4. Вставьте его в HTML.

Пример кода в `<head>`:
```html
<!-- Предварительное подключение к серверам Google для ускорения -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>

<!-- Подключение конкретного шрифта (например, Roboto) -->
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">

<!-- ВАЖНО: Ваш CSS должен идти ПОСЛЕ шрифтов -->
<link rel="stylesheet" href="style.css">
```

### Почему это хорошо?
- Браузер начинает скачивать шрифты одновременно с чтением HTML.
- Поддержка `preconnect` позволяет заранее установить соединение с сервером Google.

## 📥 Способ №2: Подключение через @import

Этот способ позволяет подключить шрифты прямо внутри вашего CSS-файла.

### Как это выглядит:
```css
/* В самом верху файла style.css */
@import url('https://fonts.googleapis.com/css2?family=Montserrat:ital,wght@0,400;0,700;1,400&display=swap');

body {
  font-family: 'Montserrat', sans-serif;
}
```

### Когда использовать?
Обычно это удобно, если у вас нет доступа к HTML-коду (например, вы работаете в CMS или конструкторе, который позволяет менять только CSS).

### Минусы:
- Это работает медленнее, чем `<link>`. Браузер сначала должен скачать CSS, прочитать его, увидеть `@import` и только потом начать качать шрифт. Получается «эффект матрешки».

## ⚡ Оптимизация: font-display

Когда вы копируете код из Google Fonts, вы видите в конце ссылки параметр `&display=swap`. Это магия, которая спасает нервы ваших пользователей.

**Как это работает:**
1. Пока файл шрифта еще качается, браузер показывает текст стандартным системным шрифтом (Arial).
2. Как только файл загрузился, браузер мгновенно «подменяет» (swap) его на красивый шрифт от Google.

Без этого параметра пользователь мог бы видеть пустой экран или невидимый текст в течение нескольких секунд («эффект невидимого текста»).

## 🎨 Использование в CSS

После того как вы подключили шрифт любым из способов, вам нужно «активировать» его в стилях:

```css
h1 {
  /* Обязательно используйте кавычки, если в названии есть пробелы */
  font-family: 'Open Sans', sans-serif;
  font-weight: 700; /* Жирное начертание, которое мы выбрали */
}

p {
  font-family: 'Open Sans', sans-serif;
  font-weight: 400; /* Обычное начертание */
}
```

## 🧩 Комбинирование шрифтов (Font Pairing)

Google Fonts позволяет подключать несколько разных семейств одной ссылкой. Дизайнеры часто используют правило: **один шрифт для заголовков, другой — для основного текста**.

```html
<link href="https://fonts.googleapis.com/css2?family=Playfair+Display:wght@700&family=Lato:wght@400&display=swap" rel="stylesheet">
```

В этом примере:
- `Playfair Display` — элегантный шрифт с засечками для заголовков.
- `Lato` — чистый шрифт без засечек для длинного текста.

> [!tip] Совет дизайнера
> Не используйте более двух разных шрифтовых семейств на одном сайте. Это делает дизайн неопрятным и перегружает страницу.

## 🛡️ Конфиденциальность и GDPR

В последние годы в Европе (и не только) возникли вопросы к Google Fonts из-за того, что при загрузке шрифта Google получает IP-адрес пользователя. Если вы работаете над проектом, где требования к приватности экстремально высоки, вы можете скачать шрифты с Google Fonts и хранить их на своем сервере. Как это сделать технически — мы разберем в следующем уроке про `@font-face`.

## ❓ Часто задаваемые вопросы

### «Что если Google Fonts перестанет работать?»
Это случается крайне редко. Сервера Google — одни из самых стабильных в мире. Однако, именно для этого мы всегда указываем «запасной» шрифт в `font-family` (например, `sans-serif`).

### «Зачем нужны цифры 400, 700 в коде?»
Это обозначение веса (жирности) шрифта:
- 100–300: Тонкие (Light)
- 400: Обычный (Regular)
- 500–600: Средний (Medium/Semi-bold)
- 700: Жирный (Bold)
- 900: Очень жирный (Black)

### «Нужно ли платить за Google Fonts?»
Нет. Все шрифты в библиотеке бесплатны для личного и коммерческого использования по лицензии SIL Open Font License или Apache License.

### «Как понять, поддерживает ли шрифт русский язык?»
В фильтрах на сайте Google Fonts выберите **Language -> Cyrillic**. Список обновится, и вы увидите только те шрифты, в которых есть русские буквы.

## 🧪 Практика: Подключаем идеальную пару

Давайте создадим стильную страницу, используя два шрифта от Google.

```html
<!-- 1. Вставляем в head -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Oswald:wght@500&family=Quicksand:wght@400;700&display=swap" rel="stylesheet">
```

```css
/* 2. Применяем в CSS */
h1 {
  font-family: 'Oswald', sans-serif;
  text-transform: uppercase;
}

body {
  font-family: 'Quicksand', sans-serif;
  font-size: 18px;
}

.bold-text {
  font-weight: 700;
}
```

## 🔑 Ключевые мысли урока

> [!abstract] Запомни
> 1. **Google Fonts** — это бесплатный и стандартный способ использования веб-шрифтов.
> 2. Метод **`<link>`** в HTML предпочтительнее, чем `@import` в CSS.
> 3. Выбирайте только те **начертания**, которые реально используете.
> 4. Не забывайте про **кириллицу** в фильтрах поиска.
> 5. Параметр **`display=swap`** делает загрузку сайта визуально быстрее.
> 6. Использование более **двух разных шрифтов** на странице — признак плохого тона.

Однако, полагаться только на сторонние сервисы не всегда правильно. Иногда дизайнеры присылают нам файлы шрифтов (в форматах `.woff2`, `.ttf`), которых нет в Google.
