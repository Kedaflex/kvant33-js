## 📖 Введение: Искусство краткости

Вы уже знаете, что для настройки текста нам нужно много свойств: семейство шрифта, размер, жирность, наклон, высота строки... Если прописывать их по отдельности для каждого элемента, наш CSS-файл быстро превратится в «простыню» из сотен строк.

CSS предлагает нам «супер-свойство» `font`. Оно позволяет объединить до шести различных свойств в одну короткую и понятную строку. Это экономит время, уменьшает вес файла и делает код более читабельным для профессионалов. Но у этого свойства есть строгие правила, которые нельзя нарушать. Сегодня мы разберем анатомию этого свойства и научимся использовать его как профи.

---

## 🛠️ Синтаксис свойства font

Представьте, что у нас есть такой набор правил:
```css
h2 {
  font-style: italic;
  font-variant: small-caps;
  font-weight: bold;
  font-size: 24px;
  line-height: 1.5;
  font-family: Arial, sans-serif;
}
```

С помощью свойства `font` мы можем превратить эти 6 строк в одну:
```css
h2 {
  font: italic small-caps bold 24px/1.5 Arial, sans-serif;
}
```

### Порядок значений (это критически важно!):
Свойство `font` работает только в определенной последовательности. Если вы перепутаете местами размер и семейство, или поставите жирность в конец — браузер просто проигнорирует всё правило.

**Эталонная последовательность:**
1. **`font-style`** (опционально): `italic`, `oblique`.
2. **`font-variant`** (опционально): чаще всего используется `small-caps` (малые прописные).
3. **`font-weight`** (опционально): `bold`, `300`, `900` и т.д.
4. **`font-size` (ОБЯЗАТЕЛЬНО)**: размер шрифта (px, rem, em).
5. **`line-height`** (опционально): идет строго через слеш `/` после размера.
6. **`font-family` (ОБЯЗАТЕЛЬНО)**: список шрифтов через запятую.

> [!warning] Главное правило
> Если вы не укажете **размер** или **семейство шрифта**, вся строка `font` будет проигнорирована браузером как ошибочная. Это два столпа, на которых держится это свойство.

---

## 🕵️ Что такое font-variant: small-caps?

Мы не разбирали это свойство отдельно, но оно часто встречается в сокращенной записи. Оно превращает все строчные буквы в заглавные, но уменьшенного размера. Это выглядит очень изысканно и часто используется в типографике для оформления подзаголовков или вступительных слов (буквиц).

В сокращенной записи оно ставится сразу после стиля (italic).

---

## 📐 Пара font-size / line-height

Это самое удобное и часто используемое сочетание в сокращенной записи. Мы привыкли, что размер и интервал неразрывно связаны. В свойстве `font` они записываются вместе через косую черту (слеш):

```css
/* Размер 16px, высота строки 1.5 (24px) */
font: 16px/1.5 sans-serif;

/* Размер 2rem, высота строки normal */
font: 2rem sans-serif;
```

Если вы не поставите слеш и второе число, браузер решит, что `line-height` должен быть `normal` (сбросит ваше предыдущее значение, если оно было).

---

## ⚠️ Коварный сброс значений

У сокращенных свойств в CSS (таких как `font`, `background`, `border`) есть одна особенность, на которой «ловят» новичков. Когда вы используете `font`, все свойства, которые вы **не указали** в строке, автоматически сбрасываются до своих значений по умолчанию.

**Пример ошибки:**
```css
p {
  font-weight: bold; /* Мы сделали текст жирным */
  /* А потом решили поменять только шрифт и размер через сокращение */
  font: 14px Arial, sans-serif;
}
```
**Результат:** Текст **перестанет быть жирным**. Почему? Потому что в строке `font` мы не указали `bold`, и CSS послушно установил `font-weight` в значение по умолчанию (`normal`).

**Как этого избежать?**
Либо указывайте абсолютно все нужные параметры внутри `font`, либо дописывайте специфические свойства **после** сокращенной записи:
```css
p {
  font: 14px Arial, sans-serif;
  font-weight: bold; /* Теперь это сработает, так как идет позже */
}
```

## 💎 Системные шрифты через font

У свойства `font` есть еще одна редкая, но полезная возможность. Вы можете использовать ключевые слова, чтобы ваш текст выглядел точно так же, как в интерфейсе операционной системы пользователя.

- `font: caption;` — шрифт для кнопок и элементов управления.
- `font: icon;` — шрифт для подписей к иконкам в системе.
- `font: menu;` — шрифт выпадающих системных меню.
- `font: message-box;` — шрифт диалоговых окон.

Сегодня это используется редко (в основном в веб-приложениях, которые мимикрируют под нативные программы), так как дизайнеры предпочитают полный контроль над стилем. Однако полезно знать, что такая возможность существует.

## 🧪 Когда использовать сокращение, а когда — нет?

### Использовать стоит:
- **В начале проекта**: Когда вы задаете базовые стили для `body` (размер, шрифт, интервал).
- **Для сложных блоков**: Когда у элемента меняется сразу 3 и более параметров шрифта.
- **Для оптимизации**: Когда вы хотите сделать код максимально компактным и профессиональным.

### Лучше писать раздельно:
- **Для состояний**: Когда вам нужно изменить только один параметр (например, только `font-weight: bold` при наведении курсора на ссылку). Использование всего свойства `font` в `:hover` — плохая практика.
- **В компонентах**: Если вы боитесь случайно сбросить другие важные настройки шрифта, заданные родителем.

## ❓ Часто задаваемые вопросы

### «Можно ли написать font: 20px Bold?»
Нет. Порядок важен: жирность (`bold`) должна идти **перед** размером (`20px`). Правильно: `font: bold 20px sans-serif;`.

### «Входит ли сюда text-align или color?»
Нет! Это самая частая ошибка. Свойство `font` объединяет только те свойства, которые начинаются на слово `font-` (плюс `line-height`). Цвет текста (`color`) и его выравнивание (`text-align`) всегда пишутся отдельно.

### «Можно ли использовать несколько шрифтов в font?»
Да, конечно. Всё, что идет после размера — это ваш привычный шрифтовой стек через запятую:
`font: 1rem/1.4 "Open Sans", Helvetica, Arial, sans-serif;`

## 🧪 Практика: Оптимизируем CSS-код

Давайте возьмем типичный «грязный» код и сделаем его аккуратным.

**До (длинно и избыточно):**
```css
.card-price {
  font-family: "Montserrat", sans-serif;
  font-size: 1.5rem;
  font-weight: 900;
  line-height: 1.2;
  font-style: normal;
  color: #e67e22;
}
```

**После (профессионально и лаконично):**
```css
.card-price {
  font: 900 1.5rem/1.2 "Montserrat", sans-serif;
  color: #e67e22;
}
```
Результат на экране будет абсолютно идентичен, но ваш CSS-файл стал в три раза короче и чище!

## 🔑 Ключевые мысли урока

> [!abstract] Запомни
> 1. **`font`** — это мощное сокращение для 6 свойств шрифта.
> 2. **Обязательные параметры**: `font-size` и `font-family`. Без них свойство не будет работать.
> 3. **Порядок критичен**: стиль, начертание и жирность идут ДО размера, шрифт — ПОСЛЕ.
> 4. Высота строки (`line-height`) пишется сразу после размера через косую черту: **`16px/1.5`**.
> 5. Помните о **сбросе значений**: не указанные в списке параметры вернутся к стандартным.
> 6. Сокращенная запись — признак опытного разработчика, который заботится о чистоте и производительности кода.
