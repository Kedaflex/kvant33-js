## 📖 Введение: Воздух между буквами

Типографика — это не только сами буквы, но и пустота между ними. В профессиональном дизайне это называется «белым пространством». Правильное использование этого пространства может сделать текст либо легким и воздушным, либо плотным и напряженным.

Представьте, что буквы в слове — это люди в очереди. Если они стоят слишком близко, им неуютно. Если слишком далеко — связь между ними теряется. Сегодня мы научимся управлять этим «социальным дистанцированием» букв и слов в CSS.

---

## 🔡 Свойство letter-spacing (Трекинг)

Свойство `letter-spacing` устанавливает интервал между символами текста. Оно добавляет (или убирает) пространство после каждого знака.

### Значения:
1. **normal**: Расстояние по умолчанию для данного шрифта.
2. **Длина (px, em, rem)**: Конкретное значение отступа.
   - *Положительное*: буквы раздвигаются.
   - *Отрицательное*: буквы сближаются.

### Почему EM — лучший выбор для интервалов?
В типографике межбуквенное расстояние всегда должно быть пропорционально размеру шрифта. Если вы укажете `letter-spacing: 2px`, то для мелкого текста это будет огромная дыра, а для гигантского заголовка — незаметный сдвиг.

Используя `em`, вы привязываете интервал к размеру буквы:
```css
h1 {
  font-size: 3rem;
  letter-spacing: 0.05em; /* Интервал будет расти вместе с заголовком */
}
```

---

## 🎨 Когда использовать letter-spacing?

### 1. Текст в верхнем регистре (UPPERCASE)
Это главное правило. Заглавные буквы в большинстве шрифтов спроектированы так, что они стоят слишком тесно друг к другу. Чтобы «капс» выглядел эстетично и легко читался, ему **всегда** нужно добавлять немного воздуха.
```css
.button {
  text-transform: uppercase;
  letter-spacing: 1px; /* Или 0.1em */
}
```

### 2. Крупные заголовки
В очень больших заголовках (display fonts) буквы могут казаться слишком разрозненными. Иногда дизайнеры намеренно немного уменьшают интервал, чтобы заголовок выглядел более цельным и «сбитым».
```css
.hero-title {
  font-size: 5rem;
  letter-spacing: -0.02em; /* Легкое сужение */
}
```

### 3. Мелкий текст
Если вам нужно написать что-то очень мелко (например, подпись в подвале сайта), буквы могут начать сливаться. Добавление `0.02em` или `0.05em` поможет глазу лучше различать символы.

---

## 📑 Свойство word-spacing

Это свойство работает точно так же, как и предыдущее, но меняет расстояние только между **словами** (увеличивает или уменьшает ширину пробела).

### Использование:
```css
p {
  word-spacing: 0.2rem;
}
```

**Когда это нужно?**
Честно говоря, в повседневной верстке `word-spacing` используется крайне редко. Шрифтовые дизайнеры уже заложили оптимальную ширину пробела в сам шрифт. Однако оно может пригодиться:
- В декоративных целях (например, в стихах или художественных заголовках).
- Чтобы визуально «разредить» текст, если слова кажутся слишком плотными (характерно для некоторых специфических шрифтов).

---

## 🧪 Опасности и ошибки

### 1. Перебор с разрядкой
Если вы сделаете `letter-spacing` слишком большим, слово распадется на отдельные буквы. Пользователю придется не «читать» слово целиком, а «собирать» его как пазл. Это очень утомляет мозг.

### 2. Слишком тесный текст
Отрицательное значение `letter-spacing` может привести к тому, что буквы начнут накладываться друг на друга. Особенно это критично для кириллицы с её сложными формами (буквы `ж`, `ш`, `щ`, `м`).

### 3. Игнорирование особенностей шрифта
Некоторые современные шрифты (например, *Inter* или *Montserrat*) уже имеют отличный баланс пространства. Прежде чем менять интервалы, посмотрите на текст свежим взглядом — возможно, он и так идеален?

---

## 🔍 Разница между кернингом и трекингом

Иногда начинающие разработчики путают эти понятия:
- **Кернинг (Kerning)**: Расстояние между *конкретной парой* букв (например, между 'A' и 'V', чтобы они не казались слишком далекими из-за своих форм). В CSS это управляется свойством `font-kerning`, которое обычно включено по умолчанию.
- **Трекинг (Tracking)**: Равномерное изменение расстояния между *всеми* буквами в блоке текста. Это и есть наше свойство `letter-spacing`.

## ❓ Часто задаваемые вопросы

### «Можно ли использовать проценты для letter-spacing?»
Нет, свойство `letter-spacing` не принимает проценты. Используйте `em` — это даст тот же эффект относительности.

### «Влияет ли letter-spacing на ширину блока?»
Да. Поскольку каждая буква отодвигается от соседней, общая длина строки увеличивается. Это важно учитывать, если ваш текст находится в блоке с фиксированной шириной — он может начать переноситься раньше времени.

### «Как сбросить letter-spacing до стандарта?»
Используйте значение `normal`. Это лучше, чем писать `0`, так как `normal` учитывает настройки самого файла шрифта.

## 🧪 Практика: Создаем эффектный баннер

```html
<div class="banner">
  <p class="overline">Новая коллекция</p>
  <h1 class="brand-name">MINIMALISM</h1>
  <p class="slogan">Простота в деталях</p>
</div>
```

```css
.banner {
  text-align: center;
  padding: 100px 20px;
  background-color: #f9f9f9;
  font-family: 'Montserrat', sans-serif;
}

.overline {
  text-transform: uppercase;
  font-size: 0.8rem;
  font-weight: 600;
  color: #888;
  /* Добавляем воздух для мелкого капса */
  letter-spacing: 0.3em;
  margin-bottom: 20px;
}

.brand-name {
  font-size: 4rem;
  font-weight: 900;
  color: #222;
  /* Чуть-чуть сужаем для мощи */
  letter-spacing: -0.05em;
  margin: 0;
}

.slogan {
  font-size: 1.2rem;
  font-style: italic;
  color: #555;
  /* Небольшое увеличение интервала между словами */
  word-spacing: 0.5rem;
}
```

## 🔑 Ключевые мысли урока

> [!abstract] Запомни
> 1. **`letter-spacing`** меняет интервал между всеми символами.
> 2. Используйте **`em`** вместо пикселей для консистентности при изменении размера шрифта.
> 3. Текст в **`uppercase`** (все заглавные) почти всегда требует увеличения интервала.
> 4. Крупные заголовки иногда требуют отрицательного `letter-spacing` для плотности.
> 5. **`word-spacing`** меняет расстояние между словами, используется реже.
> 6. Не злоупотребляйте интервалами: читабельность важнее «красоты».
