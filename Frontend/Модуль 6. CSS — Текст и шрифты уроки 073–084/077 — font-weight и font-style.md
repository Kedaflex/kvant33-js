# 077 — font-weight и font-style

- Глубоко изучить свойство `font-weight` и систему числовых значений насыщенности шрифта

## 📖 Введение: Характер и акценты

Текст — это не просто набор букв. В нем есть иерархия. Что-то мы хотим выделить как очень важное, что-то — как второстепенное, а что-то — как цитату или авторское примечание.

Если `font-size` управляет «громкостью» текста, то `font-weight` (жирность) и `font-style` (стиль) управляют его «интонацией». Жирный шрифт звучит уверенно и веско, курсив — мягко и элегантно. Сегодня мы научимся правильно использовать эти инструменты, чтобы ваш текст «зазвучал».

---

## 💪 Свойство font-weight (Насыщенность)

`font-weight` определяет, насколько толстыми будут линии букв. В CSS существует два способа задать это значение.

### 1. Ключевые слова
- `normal`: Обычный шрифт (соответствует числу 400).
- `bold`: Жирный шрифт (соответствует числу 700).
- `lighter` / `bolder`: Делает шрифт на один шаг светлее или темнее относительно родительского элемента (используется редко).

### 2. Числовая шкала (100–900)
Это более профессиональный и точный способ. Стандартная шкала выглядит так:
- **100**: Thin (Hairline) — очень тонкий.
- **200**: Extra Light (Ultra Light).
- **300**: Light — легкий.
- **400**: **Regular** — обычный.
- **500**: Medium — средний.
- **600**: Semi Bold (Demi Bold) — полужирный.
- **700**: **Bold** — жирный.
- **800**: Extra Bold (Ultra Bold).
- **900**: Black (Heavy) — очень жирный.

> [!warning] Важное напоминание
> Числовые значения будут работать только в том случае, если вы **подключили** соответствующие файлы шрифтов через Google Fonts или `@font-face`. Если вы подключили только вес 400 и 700, а в CSS написали `font-weight: 300`, браузер «округлит» значение до ближайшего доступного.

## 🎭 Свойство font-style (Стиль)

Это свойство отвечает за наклон символов.

### Значения:
1. **normal**: Стандартный прямой шрифт.
2. **italic**: Курсивное начертание. Это не просто наклоненный текст, это **отдельно нарисованные буквы**, которые часто имеют более округлые формы, напоминающие рукописный ввод.
3. **oblique**: Наклонный шрифт. Это просто математически наклоненные буквы основного шрифта.

### Иерархия выбора
В большинстве случаев дизайнеры используют `italic`. Если у шрифта нет специальной курсивной версии, браузер может попытаться сымитировать наклон (сделать `oblique` автоматически).

```css
cite {
  font-style: italic;
  color: #555;
}
```

## 🧪 Что такое "синтетический" шрифт?

Это ситуация, которой стоит избегать любому хорошему верстальщику.

Если вы указали в CSS `font-weight: bold`, но **не подключили** файл жирного шрифта, браузер не сдастся. Он возьмет обычный шрифт и «размажет» его, чтобы он казался толще. То же самое с курсивом — браузер просто наклонит буквы.

**Почему это плохо?**
1. Это выглядит некрасиво и неопрятно (теряются пропорции букв).
2. Читабельность такого текста гораздо хуже.
3. Истинный курсив и истинный жирный шрифт создаются шрифтовыми дизайнерами годами, чтобы каждая буква была идеальной. Браузерная имитация — это «дешевая подделка».

> [!tip] Совет
> Всегда проверяйте во вкладке Network или через расширения, все ли нужные веса шрифта загрузились.

## 🧱 Комбинирование весов для иерархии

Хороший дизайн часто строится на контрасте.
- Используйте **Light (300)** для вступительных абзацев (лид-текст).
- Используйте **Regular (400)** для основного чтения.
- Используйте **Medium (500)** или **Semi-bold (600)** для подзаголовков или выделения важных слов в тексте.
- Используйте **Bold (700)** для основных заголовков.
- Используйте **Black (900)** только для очень крупных, плакатных заголовков.

```css
.card-title {
  font-weight: 600; /* Достаточно заметно, но не перегружено */
}

.card-description {
  font-weight: 400; /* Максимальная читабельность */
}
```

## 📐 Тонкая настройка: font-synthesis

Если вы принципиально не хотите, чтобы браузер «дорисовывал» жирность или наклон там, где их нет, вы можете это запретить:

```css
body {
  font-synthesis: none; /* Запрещает браузеру имитировать начертания */
}
```
В этом случае, если жирного шрифта нет, текст останется обычным, даже если стоит `font-weight: bold`. Это честный подход: лучше показать обычный качественный шрифт, чем уродливую подделку.

## ❓ Часто задаваемые вопросы

### «Есть ли разница между тегом <b> и свойством font-weight: bold?»
С точки зрения внешнего вида — нет. Но с точки зрения семантики:
- `<b>` — просто делает текст жирным визуально.
- `<strong>` — сообщает поисковикам и скринридерам, что этот текст важен.
- `font-weight` — это инструмент оформления в CSS.
Профессионалы используют семантические теги (`strong`, `em`) для смысла, а CSS — для управления тем, как именно этот смысл выглядит.

### «Можно ли написать font-weight: 750?»
В современных браузерах для так называемых **вариативных шрифтов** (Variable Fonts) — да! Вы можете указать любое число от 1 до 1000. Но для обычных (статичных) шрифтов используются только «сотни» (400, 500 и т.д.).

### «Почему мой жирный шрифт выглядит одинаково с обычным?»
Скорее всего, вы забыли его подключить. Проверьте ваш `@import` или `<link>` из Google Fonts — указан ли там нужный вес?

## 🧪 Практика: Создаем карточку товара с иерархией

```html
<div class="product-card">
  <span class="category">Электроника</span>
  <h2 class="title">Беспроводные наушники</h2>
  <p class="description">Погрузитесь в мир чистого звука с нашей новой моделью...</p>
  <div class="price">12 990 ₽</div>
</div>
```

```css
.product-card {
  font-family: 'Montserrat', sans-serif; /* Допустим, мы его подключили */
  border: 1px solid #ddd;
  padding: 20px;
  max-width: 300px;
}

.category {
  font-weight: 500;
  font-style: italic;
  font-size: 0.8rem;
  color: #888;
  text-transform: uppercase;
}

.title {
  font-weight: 700;
  margin: 10px 0;
  color: #222;
}

.description {
  font-weight: 400;
  line-height: 1.5;
  color: #444;
}

.price {
  font-weight: 900;
  font-size: 1.4rem;
  color: #e67e22;
  margin-top: 15px;
}
```

## 🔑 Ключевые мысли урока

> [!abstract] Запомни
> 1. **`font-weight`** управляет жирностью текста (от 100 до 900).
> 2. **400** — это стандарт (`normal`), **700** — это жирный (`bold`).
> 3. **`font-style`** управляет наклоном, чаще всего используется **`italic`**.
> 4. Не используйте веса и стили, которые вы не подключили — это создает «синтетические» некрасивые начертания.
> 5. **Иерархия весов** помогает пользователю быстрее сканировать страницу взглядом.
> 6. Для вариативных шрифтов можно использовать дробные и точные значения (например, 550).
