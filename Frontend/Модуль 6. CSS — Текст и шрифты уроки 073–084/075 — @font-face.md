## 📖 Введение: Полная независимость

Google Fonts — это прекрасно, но профессиональный разработчик не должен зависеть от одного сервиса. Представьте ситуацию:
1. Вы работаете над корпоративным сайтом, и дизайнер создал уникальный шрифт специально для бренда. Его нет в открытых библиотеках.
2. Вам нужно, чтобы сайт работал во внутренней сети компании (интранет) без доступа к внешнему интернету.
3. Вы хотите выжать максимум производительности и избавиться от лишних запросов к сторонним серверам.

В таких случаях на помощь приходит директива **`@font-face`**. Это мощный инструмент CSS, который позволяет вам сказать браузеру: «Эй, если тебе нужен этот шрифт — возьми его в этой папке на моем сервере».

## 📦 Форматы шрифтов: Что выбрать?

Шрифты — это файлы. И у них есть разные расширения, каждое из которых имеет свою историю и степень сжатия.

1. **WOFF2 (Web Open Font Format 2.0)**: **Золотой стандарт**. Это самый современный формат с лучшим сжатием. Он на 30% легче, чем обычный WOFF.
   - *Поддержка*: Все современные браузеры.
2. **WOFF (Web Open Font Format)**: Предшественник WOFF2. Все еще актуален как запасной вариант для чуть более старых браузеров.
3. **TTF (TrueType Font)**: Старый добрый формат, привычный нам по Windows и macOS. В вебе используется редко из-за большого веса (плохое сжатие).
4. **SVG / EOT**: Древние форматы для Internet Explorer и старых iPhone. В 2024 году считаются «мёртвыми», и использовать их не нужно.

> [!tip] Правило 2024 года
> Для 99% проектов достаточно иметь шрифты в формате **WOFF2**. Если вы поддерживаете совсем старые системы — добавьте **WOFF**.

## 🛠️ Синтаксис @font-face

Директива `@font-face` пишется в самом верху вашего CSS-файла. Она не применяет шрифт к тексту, она только «регистрирует» его в системе браузера.

### Базовый пример:
```css
@font-face {
  font-family: "MyCustomFont";        /* Имя, которое мы придумали сами */
  src: url("fonts/myfont.woff2") format("woff2"); /* Путь к файлу */
  font-weight: normal;                /* Вес этого конкретного файла */
  font-style: normal;                 /* Стиль этого конкретного файла */
  font-display: swap;                 /* Ускорение отображения */
}
```

### Как это использовать:
```css
body {
  font-family: "MyCustomFont", Arial, sans-serif;
}
```

## 🧬 Регистрация разных начертаний

Это самый важный момент. Каждое начертание (жирный, курсив, тонкий) — это **отдельный файл**. Если вы хотите использовать и обычный, и жирный шрифт, вам нужно написать `@font-face` дважды, но под **одним и тем же** именем `font-family`.

```css
/* Регистрируем обычный шрифт */
@font-face {
  font-family: "BrandFont";
  src: url("fonts/brand-regular.woff2") format("woff2");
  font-weight: 400;
  font-style: normal;
}

/* Регистрируем жирный шрифт */
@font-face {
  font-family: "BrandFont";
  src: url("fonts/brand-bold.woff2") format("woff2");
  font-weight: 700;
  font-style: normal;
}

/* Регистрируем курсив */
@font-face {
  font-family: "BrandFont";
  src: url("fonts/brand-italic.woff2") format("woff2");
  font-weight: 400;
  font-style: italic;
}
```

**Почему это круто?**
Теперь в коде вам достаточно написать `font-family: "BrandFont"`. Браузер сам поймет: если у элемента стоит `font-weight: bold`, нужно взять файл `brand-bold.woff2`. Если стоит `font-style: italic`, нужно взять `brand-italic.woff2`. Это называется интеллектуальным связыванием шрифтов.

## 📂 Организация файлов в проекте

Не бросайте шрифты в корень папки. Соблюдайте порядок:

```text
my-project/
├── index.html
├── css/
│   └── style.css
└── fonts/
    ├── brand-regular.woff2
    ├── brand-bold.woff2
    └── brand-italic.woff2
```

В CSS путь будет выглядеть так: `url("../fonts/brand-regular.woff2")`. Две точки нужны, чтобы «выйти» из папки `css` и «зайти» в папку `fonts`.

## 🚀 Продвинутые техники

### Функция local()
Иногда шрифт, который вы подключаете, уже есть у пользователя (например, он дизайнер и сам его установил). Чтобы не заставлять браузер качать файл лишний раз, можно использовать `local()`.

```css
@font-face {
  font-family: "MyFont";
  src: local("MyFont"), /* Сначала ищем на компьютере */
       local("MyFont-Regular"),
       url("../fonts/myfont.woff2") format("woff2"); /* Если нет — качаем */
}
```

### Вариативные шрифты (Variable Fonts)
Это новая технология, где один файл шрифта содержит в себе сразу все веса, наклоны и ширины. Вместо 10 файлов вы качаете один (он чуть тяжелее), но можете плавно менять жирность, например, от 100 до 900.

```css
@font-face {
  font-family: 'Inter Variable';
  src: url('Inter-VariableFont_slnt,wght.woff2') format('woff2-variations');
  font-weight: 100 900; /* Поддержка диапазона весов */
}
```

## ⚠️ Частые ошибки

1. **Неправильные пути**: Проверьте консоль браузера (F12). Если там красная ошибка "404 Not Found" рядом с файлом шрифта — вы ошиблись в пути.
2. **Забытый format()**: Браузеру проще понять тип файла, если вы явно написали `format("woff2")`.
3. **Отсутствие кавычек**: Хотя для простых названий кавычки не обязательны, лучше всегда брать имя шрифта в кавычки для консистентности.
4. **Смешивание весов**: Если вы зарегистрируете жирный шрифт как `font-weight: 400`, браузер будет показывать его как обычный, и вы долго будете гадать, почему «жирность не работает».

## ❓ Часто задаваемые вопросы

### «Где брать файлы шрифтов?»
- Скачивать с Google Fonts (кнопка "Download family").
- Использовать сервисы вроде FontSquirrel или Google Webfonts Helper (отличный инструмент, который сразу генерирует `@font-face` код).
- Покупать на коммерческих сайтах (MyFonts, Adobe Fonts).

### «Нужно ли подключать TTF для поддержки Android?»
Старые версии Android требовали TTF, но современные (уже много лет как) отлично понимают WOFF/WOFF2. В 2024 году WOFF2 — ваш основной выбор.

### «Что если я скачал шрифт .otf?»
Лучше конвертировать его в .woff2 с помощью онлайн-конвертеров. OTF не оптимизирован для передачи по сети, он может весить в 10 раз больше, чем WOFF2.

## 🧪 Практика: Подключаем свой шрифт

Представьте, что мы скачали шрифт "GreatVibes" для подписи автора.

1. Кладем файл в `fonts/great-vibes.woff2`.
2. Пишем CSS:

```css
/* Регистрация */
@font-face {
  font-family: "GreatVibes";
  src: url("../fonts/great-vibes.woff2") format("woff2");
  font-weight: 400;
  font-style: normal;
  font-display: swap;
}

/* Использование */
.signature {
  font-family: "GreatVibes", cursive;
  font-size: 2.5rem;
  color: #555;
}
```

## 🔑 Ключевые мысли урока

> [!abstract] Запомни
> 1. **@font-face** позволяет использовать любые шрифты, хранящиеся на вашем сервере.
> 2. **WOFF2** — оптимальный формат для современного веба.
> 3. Каждое начертание (Bold, Italic) требует своей записи `@font-face`.
> 4. Все записи `@font-face` должны иметь одинаковое имя в **font-family**, чтобы браузер мог их переключать.
> 5. Всегда используйте **font-display: swap** для предотвращения невидимого текста.
> 6. Проверяйте пути к файлам относительно CSS-файла.
