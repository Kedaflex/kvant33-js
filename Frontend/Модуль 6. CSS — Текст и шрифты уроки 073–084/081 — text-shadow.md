## 📖 Введение: Объем и акценты

До этого момента мы работали с текстом как с плоским объектом. Но иногда дизайну нужно добавить глубины. Тень может сделать текст выпуклым, вдавливать его в фон или создавать вокруг него магическое свечение.

Однако, `text-shadow` — это не только про красоту. Это еще и мощный инструмент доступности. Например, белые буквы на светлой фотографии практически не читаются, но стоит добавить к ним едва заметную темную тень, и текст станет четким и контрастным. Сегодня мы освоим магию теней.

---

## 🛠️ Синтаксис свойства text-shadow

Свойство `text-shadow` принимает до четырех значений:

```css
h1 {
  /* смещение-X | смещение-Y | размытие | цвет */
  text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);
}
```

### Разберем подробнее:
1. **Смещение по X (Horizontal offset)**: На сколько пикселей тень сдвинется вправо (положительное число) или влево (отрицательное).
2. **Смещение по Y (Vertical offset)**: На сколько пикселей тень сдвинется вниз (положительное) или вверх (отрицательное).
3. **Радиус размытия (Blur radius)**: Чем больше число, тем мягче и прозрачнее края тени. Если поставить `0`, тень будет иметь четкие края, как у самого шрифта.
4. **Цвет (Color)**: Можно использовать любые форматы (HEX, RGB, RGBA). RGBA особенно хорош, так как позволяет делать тени полупрозрачными.

> [!tip] Факт
> В отличие от `box-shadow` (тени для блоков), у теней текста нет параметра «растяжения» (spread).

## 🚀 Продвинутая техника: Множественные тени

Вы можете добавить тексту сколько угодно теней, просто перечислив их через запятую. Это позволяет создавать очень сложные эффекты.

### Эффект контура (Outline)
С помощью четырех четких теней можно имитировать обводку букв:
```css
.outline {
  color: white;
  text-shadow:
     1px  1px 0 #000,
    -1px -1px 0 #000,
     1px -1px 0 #000,
    -1px  1px 0 #000;
}
```

### Эффект 3D-текста
Складывая тени с разным смещением, можно создать иллюзию объема:
```css
.text-3d {
  color: #f1c40f;
  text-shadow:
    0 1px 0 #c0392b,
    0 2px 0 #c0392b,
    0 3px 0 #c0392b,
    0 4px 0 #c0392b,
    0 5px 10px rgba(0,0,0,0.5);
}
```

---

## 🕯️ Эффект свечения (Neon / Glow)

Если мы не будем смещать тень (`0 0`), но зададим ей большой радиус размытия и яркий цвет, мы получим эффект неонового свечения.

```css
.neon {
  color: #fff;
  text-shadow:
    0 0 5px #fff,
    0 0 10px #fff,
    0 0 20px #ff00de,
    0 0 40px #ff00de;
}
```
*Для усиления эффекта лучше использовать темный фон.*

## 📸 Практическое применение: Текст на картинках

Это самый частый сценарий. Когда вы верстаете «шапку» сайта с фоновым изображением, вы не знаете, какого цвета будут пиксели под текстом на разных экранах.

**Плохо:** Белый текст на картинке без тени.
**Хорошо:**
```css
.hero-title {
  color: white;
  /* Мягкая темная тень делает текст читабельным на любом фоне */
  text-shadow: 0 2px 10px rgba(0, 0, 0, 0.8);
}
```
Такая тень почти незаметна глазу, но она создает необходимый контраст.

---

## 🧪 Анти-паттерны: Как не надо делать

1. **Слишком черные тени**: Чисто черная тень (`#000` без прозрачности) выглядит «грязно». Используйте полупрозрачный черный (`rgba(0,0,0,0.4)`).
2. **Огромное смещение**: Если тень улетела слишком далеко от текста, она перестает восприниматься как тень и начинает мешать чтению.
3. **Тени для мелкого текста**: Никогда не добавляйте `text-shadow` для обычных абзацев с текстом 14-16px. Это превратит чтение в пытку, так как буквы будут казаться размытыми. Тени — это инструмент для заголовков и крупных элементов.
4. **Радужные тени**: Использование слишком ярких и контрастных цветов для тени (например, зеленый на красном) — признак плохого вкуса, если только вы не делаете сайт в стиле «диско 80-х».

## ❓ Часто задаваемые вопросы

### «Влияет ли text-shadow на производительность?»
Да, если теней очень много и они сложные. Браузеру приходится пересчитывать каждый пиксель при скролле. Однако для пары заголовков на странице это абсолютно не критично.

### «Можно ли сделать внутреннюю тень (inset)?»
К сожалению, нет. Значение `inset`, которое работает для `box-shadow`, не поддерживается в `text-shadow`. Для создания эффекта «вдавленного» текста используются хитрости со светлыми и темными тенями одновременно.

### «Как сделать тень только с одной стороны?»
Просто используйте смещение по одной оси. Например, `5px 0 0 red` создаст четкую тень только справа от букв.

## 🧪 Практика: Создаем "вдавленный" заголовок (Letterpress)

Этот эффект был популярен в дизайне iOS и macOS прошлых лет. Он имитирует текст, выдавленный в металле или бумаге.

```html
<div class="press-container">
  <h1 class="letterpress">ГОЛОС СТАЛИ</h1>
</div>
```

```css
.press-container {
  background-color: #555; /* Тень работает только на цветном фоне */
  padding: 50px;
  text-align: center;
}

.letterpress {
  color: #333; /* Текст чуть темнее фона */
  font-family: sans-serif;
  font-weight: bold;
  /* Светлая тень снизу создает эффект фаски */
  text-shadow: 0 1px 1px rgba(255, 255, 255, 0.2);
}
```

## 🔑 Ключевые мысли урока

> [!abstract] Запомни
> 1. **`text-shadow`** добавляет тени буквам текста.
> 2. Параметры: **смещение-X, смещение-Y, размытие, цвет**.
> 3. Используйте **RGBA** для мягких, реалистичных теней.
> 4. **Множественные тени** позволяют создавать 3D, обводку и свечение.
> 5. Главная польза тени — повышение **контраста** на сложном фоне.
> 6. Не используйте тени для **мелкого основного текста**.
