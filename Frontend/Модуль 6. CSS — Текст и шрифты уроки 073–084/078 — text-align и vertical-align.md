## 📖 Введение: Порядок в строках

Текст может быть идеально подобран по шрифту, размеру и жирности, но если он «рассыпан» по странице хаотично — сайт будет выглядеть дилетантски. Выравнивание — это то, что создает структуру и направляет взгляд читателя.

В CSS за расположение текста отвечают два основных свойства: одно работает по горизонтали (`text-align`), другое — по вертикали (`vertical-align`). И если первое интуитивно понятно почти каждому, то второе является одним из самых «капризных» свойств в CSS, которое часто сбивает новичков с толку. Сегодня мы расставим всё по местам.

---

## ↔️ Горизонтальное выравнивание: text-align

Свойство `text-align` определяет, как строки текста выравниваются внутри своего контейнера (блочного элемента).

### Основные значения:

1. **left**: Текст прижат к левому краю. Стандарт для языков с письмом слева направо (русский, английский). Создает «рваный» правый край, который легко читать.
2. **right**: Текст прижат к правому краю. Используется для подписей, дат или в специфических элементах интерфейса.
3. **center**: Каждая строка центрируется. Идеально для коротких заголовков или цитат.
4. **justify**: Текст растягивается по всей ширине контейнера. Браузер автоматически увеличивает пробелы между словами.
   - *Осторожно*: В вебе `justify` часто выглядит плохо, так как браузеры не всегда умеют правильно расставлять переносы, что приводит к «дырам» в тексте.

### Современные значения (Logical Properties):
- **start**: Выравнивание по началу строки (слева для русского, справа для арабского).
- **end**: Выравнивание по концу строки.
Это часть концепции «логических свойств», которая делает верстку универсальной для любого языка.

> [!warning] Важно
> `text-align` задается **родителю** (блоку), а не самому тексту. Если вы напишете `text-align: center` для ссылки `<a>`, ничего не произойдет, пока вы не зададите это её родителю-абзацу или не сделаете ссылку блочной.

## ↕️ Вертикальное выравнивание: vertical-align

Это свойство окутано мифами. Многие новички пытаются использовать его, чтобы центрировать текст внутри огромного `div`, и... у них ничего не получается.

**Главный секрет**: `vertical-align` работает только для:
- **Строчных элементов** (inline, inline-block).
- **Ячеек таблиц** (table-cell).

Оно выравнивает элемент относительно базовой линии (baseline) его родителя или относительно строки.

### Популярные значения:

1. **baseline**: (По умолчанию). Элемент выравнивается по базовой линии родителя (воображаемая линия, на которой стоят буквы без «хвостов»).
2. **top**: Верхний край элемента выравнивается по самому высокому элементу в строке.
3. **bottom**: Нижний край элемента выравнивается по самому низкому элементу в строке.
4. **middle**: Элемент выравнивается по середине высоты строчных букв (x-height) родителя. Часто используется для выравнивания иконок рядом с текстом.
5. **sub / super**: Сдвигает элемент как нижний или верхний индекс (аналог тегов `<sub>` и `<sup>`).

### Использование в таблицах
В таблицах `vertical-align` работает предсказуемо и удобно. Оно определяет, где будет контент внутри ячейки: сверху, снизу или посередине.

## 📐 Проблема "загадочного отступа" под картинкой

Вы наверняка замечали: если положить картинку `<img>` внутрь `div`, внизу остается странная полоска в пару пикселей, даже если убрать все отступы.

**Почему это происходит?**
Изображение по умолчанию является `inline` элементом и стоит на `baseline`. А под базовой линией всегда остается место для «хвостов» букв (десцендеров), таких как у букв `ц`, `щ`, `у`, `p`, `g`.

**Как лечить?**
Просто измените выравнивание картинки:
```css
img {
  vertical-align: middle; /* Или top, или bottom */
}
```
После этого загадочный отступ исчезнет.

## 🎯 Практические кейсы

### 1. Центрирование иконки и текста
Если у вас есть кнопка с иконкой и текстом, и они не на одной линии:
```css
.icon {
  display: inline-block;
  vertical-align: middle;
  width: 20px;
  height: 20px;
}

.text {
  vertical-align: middle;
}
```

### 2. Красивая подпись к цитате
```css
blockquote {
  text-align: center; /* Текст цитаты по центру */
}

cite {
  display: block;
  text-align: right; /* Автор цитаты — справа */
  font-style: italic;
}
```

## ⚖️ Justify: стоит ли использовать?

В профессиональной веб-верстке `text-align: justify` — редкий гость. В отличие от печатных книг, ширина экрана в вебе постоянно меняется. На узких экранах (смартфонах) «выключка по ширине» превращает текст в решето из огромных пробелов.

Если вы всё же решили использовать `justify`, обязательно добавьте свойство для автоматических переносов:
```css
p {
  text-align: justify;
  hyphens: auto; /* Позволяет браузеру разбивать слова по слогам */
}
```
*Примечание: для работы `hyphens` в HTML обязательно должен быть указан атрибут языка: `<html lang="ru">`.*

## ❓ Часто задаваемые вопросы

### «Как центрировать целый блок, а не текст внутри него?»
`text-align: center` центрирует только **содержимое**. Чтобы центрировать сам блок (например, `div`), нужно использовать `margin: 0 auto;` (мы изучим это в модуле про блочную модель). Не путайте эти два подхода!

### «Почему vertical-align: middle не центрирует текст в кнопке?»
Потому что высота строки в кнопке может быть больше самого текста. Чтобы центрировать текст в блоке фиксированной высоты, проще всего сделать `line-height` равным высоте этого блока.

### «Что такое baseline простыми словами?»
Представьте школьную тетрадь в линейку. Линия, на которой вы пишете буквы — это и есть `baseline`. Хвостики букв «у» или «д» уходят ниже этой линии.

## 🧪 Практика: Оформляем сложную шапку

```html
<header class="hero">
  <h1>Добро пожаловать в мир CSS</h1>
  <p class="promo">Изучайте основы типографики с нами</p>
  <div class="actions">
    <a href="#" class="btn">Начать <img src="arrow.svg" alt="" class="icon"></a>
  </div>
</header>
```

```css
.hero {
  background-color: #f4f4f4;
  padding: 60px;
  text-align: center; /* Центрируем всё содержимое заголовка */
}

.hero h1 {
  text-align: inherit; /* Наследует center от родителя */
  color: #2c3e50;
}

.promo {
  text-align: center;
  max-width: 500px;
  margin-left: auto;
  margin-right: auto; /* Центрируем сам блок параграфа */
}

.btn .icon {
  width: 16px;
  vertical-align: middle; /* Иконка на одной линии с текстом */
  margin-left: 8px;
}
```

## 🔑 Ключевые мысли урока

> [!abstract] Запомни
> 1. **`text-align`** выравнивает контент внутри блока по горизонтали.
> 2. Свойство **`text-align`** задается контейнеру (родителю), а не самому тексту.
> 3. **`justify`** требует осторожности и использования переносов (`hyphens`).
> 4. **`vertical-align`** работает только для строчных элементов (`inline`) и ячеек таблиц.
> 5. Значение **`vertical-align: middle`** — лучший друг при выравнивании иконок с текстом.
> 6. Не забывайте про **baseline**, если видите странные отступы под изображениями.
