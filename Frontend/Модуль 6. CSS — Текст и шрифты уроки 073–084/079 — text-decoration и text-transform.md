## 📖 Введение: Декор и регистр

В мире типографики детали решают всё. Подчеркнутое слово сразу считывается как ссылка. Перечеркнутая цена в интернет-магазине кричит о скидке. Заголовки, написанные только заглавными буквами, выглядят более монументально и важно.

Сегодня мы изучим, как CSS позволяет нам менять эти визуальные характеристики текста. Мы узнаем, как сделать подчеркивание красивым и современным, и как управлять регистром букв так, чтобы не переписывать текст в HTML-файле.

---

## 🎨 Свойство text-decoration

Традиционно мы использовали это свойство для одного — чтобы убрать подчеркивание у ссылок (`text-decoration: none`). Но современный CSS превратил его в мощный инструмент декорирования.

### Основные типы линий:
- **underline**: Подчеркивание (стандарт для ссылок).
- **overline**: Линия над текстом (используется редко).
- **line-through**: Перечеркивание (используется для удаленного контента или старых цен).
- **none**: Убирает любые линии.

### Современный составной синтаксис:
Теперь мы можем управлять линией гораздо точнее:
```css
a {
  /* Тип линии, стиль (волнистая), цвет, толщина */
  text-decoration: underline wavy orange 2px;
}
```

### Отдельные свойства (для гибкости):
1. **text-decoration-line**: Что именно рисуем (`underline`, `line-through`).
2. **text-decoration-color**: Какого цвета линия.
3. **text-decoration-style**: Какая линия (`solid` — сплошная, `double` — двойная, `dotted` — точками, `dashed` — пунктиром, `wavy` — волнистая).
4. **text-decoration-thickness**: Толщина линии (в `px`, `em` или процентах).

---

## 📏 Новинка: text-underline-offset

Долгое время подчеркивание в вебе выглядело «грязным», потому что линия проходила слишком близко к буквам и пересекала «хвосты» (например, у букв `д`, `у`, `ц`).

Свойство `text-underline-offset` позволяет отодвинуть линию ниже:
```css
.modern-link {
  text-decoration: underline;
  text-decoration-color: #3498db;
  text-underline-offset: 4px; /* Линия стала ниже и чище */
}
```
Это маленькое свойство кардинально улучшает внешний вид ссылок в тексте.

## 🔠 Свойство text-transform

Это свойство позволяет менять регистр букв (заглавные/строчные) чисто визуально, не меняя сам текст в HTML.

**Зачем это нужно?**
Представьте, что в базе данных имена пользователей хранятся как попало: «иван», «МАРИЯ», «алексей». Вы можете привести их к единому красивому виду с помощью CSS.

### Значения:
1. **uppercase**: ВСЕ БУКВЫ СТАНОВЯТСЯ ЗАГЛАВНЫМИ. Часто используется для кнопок, меню и важных заголовков.
2. **lowercase**: все буквы становятся строчными. Иногда используется в минималистичном дизайне.
3. **capitalize**: Каждое Слово Начинается С Большой Буквы. Идеально для имен, названий товаров или заголовков статей.
4. **none**: Оставляет текст в том виде, в котором он написан в HTML.

```css
.main-menu a {
  text-transform: uppercase;
  letter-spacing: 1px; /* При капсе всегда стоит добавлять немного межбуквенного интервала */
}
```

> [!warning] Важно
> `text-transform` меняет только **отображение**. Если пользователь скопирует текст, он может скопироваться в оригинальном регистре (зависит от браузера). Также помните, что текст, написанный только заглавными буквами, читается на 10-15% медленнее, чем обычный.

## 🧪 Практические советы по использованию

### 1. Подчеркивание — только для ссылок
Это золотое правило веба. Пользователи привыкли: если текст подчеркнут — на него можно нажать. Не подчеркивайте обычный текст для выделения, используйте `bold` или `italic`.

### 2. Скидки и цены
```css
.old-price {
  text-decoration: line-through;
  text-decoration-color: red;
  color: #888;
  font-size: 0.9em;
}
```

### 3. "Умные" заголовки
Если вы хотите, чтобы ваши заголовки всегда выглядели аккуратно, используйте `capitalize`. Даже если контент-менеджер забудет нажать Shift, CSS всё исправит.

## ♿ Доступность и декор

Некоторые дизайнеры любят убирать подчеркивание у ссылок внутри текста (`text-decoration: none`).

**Почему это плохо?**
Люди с нарушениями цветовосприятия (дальтоники) могут не отличить синюю ссылку от черного текста. Подчеркивание — это дополнительный визуальный сигнал, который делает ваш сайт доступным для всех.

**Компромисс:**
Можно сделать подчеркивание очень тонким или светлым, либо убирать его в обычном состоянии и плавно показывать при наведении (`:hover`).

## ❓ Часто задаваемые вопросы

### «Можно ли сделать подчеркивание пунктиром?»
Да: `text-decoration-style: dashed;`. Это часто используется для ссылок, которые не ведут на другую страницу, а открывают всплывающее окно или подсказку.

### «Почему capitalize не работает для русского языка?»
Оно работает! Но браузер ориентируется на пробелы. Если слова написаны через дефис (например, «кое-кто»), браузер может сделать заглавной только первую букву первого слова.

### «В чем разница между text-decoration и border-bottom?»
`text-decoration` — это часть шрифта. Она учитывает наклон курсива, переносится вместе со строками и не влияет на размеры блоков. `border-bottom` — это рамка блока. Она всегда прямая, находится ниже и может влиять на верстку. Для ссылок в тексте лучше использовать `text-decoration`.

## 🧪 Практика: Оформляем карточку статьи

```html
<div class="article-card">
  <span class="badge">Новинка</span>
  <h3>путешествие по исландии</h3>
  <p>Узнайте о самых красивых водопадах и вулканах...</p>
  <a href="#" class="more-link">Читать далее</a>
</div>
```

```css
.article-card {
  font-family: sans-serif;
  padding: 20px;
  border: 1px solid #eee;
}

.badge {
  text-transform: uppercase;
  font-weight: bold;
  font-size: 10px;
  background: #000;
  color: #fff;
  padding: 2px 6px;
}

.article-card h3 {
  text-transform: capitalize; /* "путешествие по исландии" -> "Путешествие По Исландии" */
  color: #333;
}

.more-link {
  color: #3498db;
  text-decoration: underline;
  text-decoration-thickness: 1px;
  text-underline-offset: 5px;
  transition: text-decoration-color 0.3s;
}

.more-link:hover {
  text-decoration-color: #2980b9;
  text-decoration-thickness: 2px;
}
```

## 🔑 Ключевые мысли урока

> [!abstract] Запомни
> 1. **`text-decoration`** — комплексное свойство для линий вокруг текста.
> 2. Современный CSS позволяет менять **стиль, цвет и толщину** подчеркивания.
> 3. **`text-underline-offset`** делает ссылки более аккуратными и читабельными.
> 4. **`text-transform`** меняет регистр (UPPERCASE, lowercase, Capitalize) без правки HTML.
> 5. Не используйте подчеркивание для обычного текста, чтобы не путать пользователей.
> 6. Текст в верхнем регистре (`uppercase`) требует дополнительного межбуквенного расстояния.
