## 🖍️ Тег `<mark>` — выделение (маркер)

**`<mark>`** выделяет текст, который **релевантен** в текущем контексте. Визуально — жёлтый фон, как маркер-выделитель в учебнике.

```html
<p>На экзамене будут вопросы по темам: циклы, массивы и <mark>функции</mark>.</p>
```
### Как выглядит по умолчанию

|Свойство|Значение|
|---|---|
|Фон|Жёлтый (`yellow`)|
|Цвет текста|Чёрный (не меняется)|
|Шрифт|Обычный (не жирный, не курсивный)|

### Когда использовать `<mark>`

|Уместно ✅|Пример|
|---|---|
|Результаты поиска|Подсветка найденного слова в тексте|
|Ключевые слова|Выделение важного термина в учебном тексте|
|Цитата с акцентом|Выделение части цитаты, на которую обращаете внимание|

|Неуместно ❌|Что использовать вместо|
|---|---|
|Просто для красоты|CSS `background-color`|
|Для важности|`<strong>`|
|Для ударения|`<em>`|

### Примеры

```HTML
<!-- Результат поиска -->
<h2>Результаты поиска по запросу «HTML»</h2>
<p><mark>HTML</mark> — язык разметки для создания веб-страниц.</p>
<p>Изучение <mark>HTML</mark> — первый шаг в веб-разработке.</p>

<!-- Учебный текст -->
<p>Запомните: <mark>переменные, объявленные через const, нельзя переназначить</mark>.</p>

<!-- Цитата с акцентом -->
<blockquote>
  <p>Единственный способ делать великую работу — <mark>любить то, что делаешь</mark>.</p>
</blockquote>
```

> [!note] Семантика `<mark>`  
> `<mark>` не означает «важный текст» — для этого есть `<strong>`. `<mark>` означает «текст, **выделенный для справки** в данном контексте». Представь, что кто-то взял маркер и подсветил фрагмент в уже написанном тексте.

## ❌ Тег `<del>` — удалённый текст

**`<del>`** обозначает текст, который был **удалён** из документа. Визуально — зачёркнутый текст.

```HTML
<p>Цена: <del>1999 руб.</del> 999 руб.</p>
```

### Как выглядит по умолчанию

|Свойство|Значение|
|---|---|
|Оформление|~~Зачёркнутый текст~~|
|Цвет|Не меняется|
|Шрифт|Обычный|

### Атрибуты `<del>`

У `<del>` есть два специальных атрибута:

|Атрибут|Назначение|Пример|
|---|---|---|
|`cite`|URL документа, объясняющего удаление|`cite="https://example.com/changelog"`|
|`datetime`|Дата и время удаления|`datetime="2024-03-15"`|


```HTML
<p>
  Встреча состоится
  <del datetime="2024-03-10">15 марта</del>
  <ins datetime="2024-03-10">22 марта</ins>.
</p>
```

### Когда использовать `<del>`

|Уместно ✅|Пример|
|---|---|
|Старая цена (скидка)|`<del>1999 руб.</del> 999 руб.`|
|Исправления в документе|`Столица Австралии — <del>Сидней</del> <ins>Канберра</ins>`|
|Изменения в списке задач|`<del>Купить молоко</del>` (выполнено)|
|Юридические правки|Изменения в договоре|

## ➕ Тег `<ins>` — вставленный текст

**`<ins>`** обозначает текст, который был **добавлен** в документ. Визуально — подчёркнутый текст.

HTML

```
<p>Столица Австралии — <del>Сидней</del> <ins>Канберра</ins>.</p>
```

### Как выглядит по умолчанию

|Свойство|Значение|
|---|---|
|Оформление|Подчёркнутый текст|
|Цвет|Не меняется|
|Шрифт|Обычный|

### Атрибуты `<ins>`

Такие же, как у `<del>`:

|Атрибут|Назначение|Пример|
|---|---|---|
|`cite`|URL документа, объясняющего вставку|`cite="https://example.com/changelog"`|
|`datetime`|Дата и время вставки|`datetime="2024-03-15"`|

### `<del>` и `<ins>` — пара

Эти теги часто используются **вместе** — один показывает, что было, другой — что стало:

```HTML
<!-- Исправление ошибки -->
<p>Земля вращается вокруг <del>Луны</del> <ins>Солнца</ins>.</p>

<!-- Изменение цены -->
<p>Подписка: <del>599 руб./мес</del> → <ins>399 руб./мес</ins></p>

<!-- Изменение расписания -->
<p>Урок: <del>понедельник, 10:00</del> <ins>вторник, 11:00</ins></p>
```

## 🔽 Тег `<sub>` — подстрочный текст

**`<sub>`** (subscript) отображает текст **ниже** базовой линии, уменьшенным размером.

```HTML
<p>Формула воды: H<sub>2</sub>O</p>
```

**Результат:** H₂O

### Когда использовать `<sub>`

| Уместно ✅              | Пример                         | Результат |
| ---------------------- | ------------------------------ | --------- |
| Химические формулы     | `H<sub>2</sub>SO<sub>4</sub>`  | H₂SO₄     |
| Математические индексы | `x<sub>1</sub>, x<sub>2</sub>` | x₁, x₂    |
| Сноски                 | `Текст<sub>[1]</sub>`          | Текст₍₁₎  |

```HTML
<p>Серная кислота: H<sub>2</sub>SO<sub>4</sub></p>
<p>Углекислый газ: CO<sub>2</sub></p>
<p>Глюкоза: C<sub>6</sub>H<sub>12</sub>O<sub>6</sub></p>
<p>Координаты точки: (x<sub>1</sub>, y<sub>1</sub>)</p>
```

---

## 🔼 Тег `<sup>` — надстрочный текст

**`<sup>`** (superscript) отображает текст **выше** базовой линии, уменьшенным размером.

```HTML
<p>E = mc<sup>2</sup></p>
```

**Результат:** E = mc²

### Когда использовать `<sup>`

| Уместно ✅                         | Пример                           | Результат |
| --------------------------------- | -------------------------------- | --------- |
| Степени                           | `x<sup>2</sup> + y<sup>2</sup>`  | x² + y²   |
| Сноски                            | `Текст<sup>[1]</sup>`            | Текст¹    |
| Порядковые числительные (в англ.) | `1<sup>st</sup>, 2<sup>nd</sup>` | 1ˢᵗ, 2ⁿᵈ  |
| Товарные знаки                    | `Apple<sup>®</sup>`              | Apple®    |

```HTML
<p>Теорема Пифагора: a<sup>2</sup> + b<sup>2</sup> = c<sup>2</sup></p>
<p>Площадь комнаты: 25 м<sup>2</sup></p>
<p>Объём бассейна: 50 м<sup>3</sup></p>
<p>Скорость света: 3 × 10<sup>8</sup> м/с</p>
<p>Примечание<sup>[1]</sup>: см. раздел «Источники».</p>
```

---

### `<sub>` и `<sup>` вместе

```HTML
<!-- Изотопы -->
<p>Углерод-14: <sup>14</sup><sub>6</sub>C</p>

<!-- Математика -->
<p>a<sub>n</sub> = a<sub>1</sub> × r<sup>n-1</sup></p>
```

## 🔹 Тег `<small>` — мелкий шрифт (побочный комментарий)

**`<small>`** обозначает **побочный текст**: примечания, юридические оговорки, копирайт.

```HTML
<p>Скидка 50% на все товары!</p>
<p><small>Акция действует до 31 марта. Подробности у продавца.</small></p>
```

### Как выглядит по умолчанию

|Свойство|Значение|
|---|---|
|Размер шрифта|Меньше обычного (~80%)|
|Стиль|Обычный|

### Когда использовать `<small>`

|Уместно ✅|Пример|
|---|---|
|Копирайт|`<small>© 2024 Компания</small>`|
|Юридические оговорки|`<small>Предложение не является офертой</small>`|
|Мелкий шрифт в рекламе|`<small>*Подробности на сайте</small>`|
|Примечание к цене|`<small>Цена указана без НДС</small>`|

```HTML
<footer>
  <p><small>© 2024 ООО «Ромашка». Все права защищены.</small></p>
  <p><small>Использование материалов сайта без разрешения запрещено.</small></p>
</footer>
```

> [!warning] `<small>` — не для уменьшения текста  
> Если нужно просто уменьшить текст ради дизайна — используй CSS `font-size`. Тег `<small>` имеет **семантическое значение** — побочный, менее важный текст.

## 🔹 Тег `<s>` — неактуальный текст

**`<s>`** (strikethrough) обозначает текст, который **больше не актуален**, но не был удалён из документа.

```HTML
<p><s>В наличии</s> Нет в наличии</p>
```

### Разница между `<s>` и `<del>`

|Характеристика|`<s>`|`<del>`|
|---|---|---|
|Внешний вид|~~Зачёркнутый~~|~~Зачёркнутый~~|
|Семантика|Текст **больше не актуален**|Текст **был удалён** из документа|
|Контекст|Неактуальная информация|Редактирование, исправление|
|Атрибуты cite/datetime|❌ Нет|✅ Есть|

```HTML
<!-- <s> — неактуальная информация -->
<p>Вакансия: <s>Junior Frontend Developer</s> (позиция закрыта)</p>
<p><s>Осталось 3 места</s> Все места заняты</p>

<!-- <del> — редакторская правка -->
<p>Встреча <del>в понедельник</del> <ins>во вторник</ins>.</p>
```

### Простое правило

|Вопрос|Ответ|
|---|---|
|Текст **исправили** на другой?|`<del>` + `<ins>`|
|Текст **стал неактуальным**, но его оставили для контекста?|`<s>`|

## 🔹 Тег `<u>` — подчёркнутый текст

**`<u>`** (underline) обозначает текст, который визуально **подчёркнут** и семантически отличается от окружающего — например, содержит ошибку или является именем собственным (в китайском языке).

```HTML
<p>В слове «сабака» допущена <u>арфографическая</u> ошибка.</p>
```

### Когда использовать `<u>`

|Уместно ✅|Пример|
|---|---|
|Орфографическая ошибка|Подчёркивание ошибки в тексте|
|Имена собственные (в некоторых языках)|Китайская типографическая традиция|
|Аннотация|Выделение конкретного участка текста|

|Неуместно ❌|Почему|
|---|---|
|Просто для подчёркивания|Используй CSS `text-decoration: underline`|
|Для ссылок|Пользователь примет подчёркнутый текст за ссылку и будет кликать|

> [!warning] Осторожно с `<u>`!  
> В интернете **подчёркнутый текст = ссылка**. Это устоявшееся соглашение. Если ты подчёркиваешь обычный текст тегом `<u>`, пользователи будут пытаться по нему кликнуть и раздражаться, что ничего не происходит.

## 🔹 Тег `<abbr>` — аббревиатура

**`<abbr>`** обозначает **аббревиатуру** или **акроним**. Атрибут `title` раскрывает полное значение.

```HTML
<p>Для создания сайтов используют <abbr title="HyperText Markup Language">HTML</abbr>,
<abbr title="Cascading Style Sheets">CSS</abbr> и
<abbr title="JavaScript">JS</abbr>.</p>
```

### Как выглядит по умолчанию

|Браузер|Оформление|
|---|---|
|Большинство|Пунктирное подчёркивание|
|При наведении|Всплывающая подсказка с текстом из `title`|

### Примеры

```HTML
<p>Стандарты веба разрабатывает <abbr title="World Wide Web Consortium">W3C</abbr>.</p>
<p>Файлы передаются по протоколу <abbr title="File Transfer Protocol">FTP</abbr>.</p>
<p>Я работаю в <abbr title="Общество с ограниченной ответственностью">ООО</abbr> «Ромашка».</p>
```

> [!tip] Раскрывай аббревиатуру при первом использовании  
> На длинной странице достаточно обернуть аббревиатуру в `<abbr>` **один раз** — при первом появлении. Дальше можно писать без `<abbr>`.

## 🧪 Практика

Создай файл и набери вручную:

```HTML
<!DOCTYPE html>
<html lang="ru">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Форматирование текста — часть 2</title>
  </head>
  <body>

    <h1>Теги форматирования текста</h1>

    <!-- mark -->
    <h2>Выделение маркером</h2>
    <p>На экзамене обязательно будут вопросы по теме <mark>замыкания в JavaScript</mark>.</p>
    <p>В документе найдено 3 совпадения со словом «<mark>HTML</mark>».</p>

    <!-- del + ins -->
    <h2>Удаление и вставка</h2>
    <p>Цена: <del>2499 руб.</del> <ins>1799 руб.</ins></p>
    <p>Столица Казахстана — <del>Алматы</del> <ins>Астана</ins>.</p>
    <p>
      Встреча состоится
      <del datetime="2024-06-01">5 июня</del>
      <ins datetime="2024-06-01">12 июня</ins>.
    </p>

    <!-- s -->
    <h2>Неактуальная информация</h2>
    <p><s>Осталось 5 мест</s> — Все места заняты!</p>
    <p>Вакансия: <s>Frontend Developer (Junior)</s> — позиция закрыта</p>

    <!-- sub и sup -->
    <h2>Подстрочный и надстрочный текст</h2>
    <p>Вода: H<sub>2</sub>O</p>
    <p>Серная кислота: H<sub>2</sub>SO<sub>4</sub></p>
    <p>Этанол: C<sub>2</sub>H<sub>5</sub>OH</p>

    <p>Теорема Пифагора: a<sup>2</sup> + b<sup>2</sup> = c<sup>2</sup></p>
    <p>Площадь квартиры: 64 м<sup>2</sup></p>
    <p>Скорость света: 3 × 10<sup>8</sup> м/с</p>

    <p>Изотоп урана: <sup>235</sup><sub>92</sub>U</p>

    <!-- small -->
    <h2>Мелкий шрифт</h2>
    <p>Скидка 70% на всю коллекцию!</p>
    <p><small>*Акция действует с 1 по 15 марта. Скидка не суммируется с другими акциями. Подробности на кассе.</small></p>

    <!-- u -->
    <h2>Подчёркивание (ошибки)</h2>
    <p>В предложении «Корова <u>летела</u> по небу» допущена смысловая ошибка.</p>

    <!-- abbr -->
    <h2>Аббревиатуры</h2>
    <p>Каждый веб-разработчик знает <abbr title="HyperText Markup Language">HTML</abbr>, <abbr title="Cascading Style Sheets">CSS</abbr> и <abbr title="JavaScript">JS</abbr>.</p>
    <p>Стандарты разрабатывает <abbr title="World Wide Web Consortium">W3C</abbr>.</p>
    <p>Сайт размещён на <abbr title="Виртуальный выделенный сервер">VPS</abbr>.</p>

    <hr>

    <!-- Комбинации -->
    <h2>Комбинации тегов</h2>
    <p><strong>Важно!</strong> Цена снижена: <del>4999 руб.</del> <mark><ins>2499 руб.</ins></mark></p>
    <p><small>Формула массы: E = mc<sup>2</sup>, где c — скорость света.</small></p>
    <p>Самый популярный напиток в мире — <strong>H<sub>2</sub>O</strong> (вода).</p>

    <hr>
    <footer>
      <p><small>© 2024 Учебный проект. Все примеры вымышлены.</small></p>
    </footer>

  </body>
</html>
```

### Что проверить в браузере

|Что проверить|Ожидаемый результат|
|---|---|
|`<mark>`|Жёлтый фон на выделенных словах|
|`<del>`|~~Зачёркнутый~~ текст|
|`<ins>`|Подчёркнутый текст|
|`<s>`|~~Зачёркнутый~~ текст (выглядит как `<del>`)|
|`<sub>`|Текст ниже базовой линии (H₂O)|
|`<sup>`|Текст выше базовой линии (x²)|
|`<small>`|Текст меньшего размера|
|`<u>`|Подчёркнутый текст|
|`<abbr>`|Пунктирное подчёркивание, подсказка при наведении|
|Комбинации|Все теги корректно работают вместе|

|Тег|Внешний вид|Семантика|Когда использовать|
|---|---|---|---|
|`<strong>`|**Жирный**|Высокая важность|Предупреждения, важные условия|
|`<b>`|**Жирный**|Привлечение внимания|Названия, термины, лид|
|`<em>`|_Курсив_|Смысловое ударение|Интонационный акцент|
|`<i>`|_Курсив_|Альтернативный голос|Иностранные слова, мысли, названия|
|`<mark>`|Жёлтый фон|Релевантность в контексте|Результаты поиска, ключевые фразы|
|`<del>`|~~Зачёркнутый~~|Удалённый текст|Исправления, старые цены|
|`<ins>`|Подчёркнутый|Добавленный текст|Исправления, новые данные|
|`<s>`|~~Зачёркнутый~~|Неактуальный текст|Закрытые вакансии, устаревшая информация|
|`<sub>`|Подстрочный|Подстрочный индекс|Химические формулы, индексы|
|`<sup>`|Надстрочный|Надстрочный индекс|Степени, сноски, единицы измерения|
|`<small>`|Мелкий шрифт|Побочный комментарий|Копирайт, юридические оговорки|
|`<u>`|Подчёркнутый|Отличающийся текст|Ошибки в тексте (использовать с осторожностью)|
|`<abbr>`|Пунктирное подчёркивание|Аббревиатура|Сокращения с расшифровкой в `title`|

## ⚠️ Типичные ошибки

### Ошибка 1: `<s>` вместо `<del>` для правок

```HTML
<!-- ❌ Неправильно: это редакторская правка, не «неактуальность» -->
<p>Встреча <s>в понедельник</s> во вторник.</p>

<!-- ✅ Правильно -->
<p>Встреча <del>в понедельник</del> <ins>во вторник</ins>.</p>
```

---

### Ошибка 2: `<small>` для уменьшения текста ради дизайна

```HTML
<!-- ❌ Неправильно: small — не для дизайна -->
<p><small>Этот абзац просто должен быть мельче.</small></p>

<!-- ✅ Правильно -->
<p class="small-text">Этот абзац стилизован через CSS.</p>
```

```CSS
.small-text {
    font-size: 14px;
}
```

---

### Ошибка 3: `<u>` для выделения обычного текста

```HTML
<!-- ❌ Опасно: пользователь примет за ссылку -->
<p>Обратите внимание на <u>этот пункт правил</u>.</p>

<!-- ✅ Лучше -->
<p>Обратите внимание на <mark>этот пункт правил</mark>.</p>

<!-- ✅ Или через CSS с другим стилем подчёркивания -->
<p>Обратите внимание на <span class="highlight">этот пункт правил</span>.</p>
```

---
### Ошибка 4: Забытый `title` у `<abbr>`

```HTML
<!-- ❌ Бессмысленно без title -->
<p>Мы используем <abbr>CSS</abbr>.</p>

<!-- ✅ Правильно -->
<p>Мы используем <abbr title="Cascading Style Sheets">CSS</abbr>.</p>
```

Без `title` тег `<abbr>` не даёт пользователю никакой дополнительной информации.

## ❓ Частые вопросы

### «`<del>` и `<s>` выглядят одинаково. Как браузер их различает?»

Визуально — **никак**. Оба зачёркнутые. Различие **семантическое**: `<del>` — редакторское удаление (был текст A, стал текст B), `<s>` — неактуальная информация (было актуально, теперь нет). Экранные читалки и поисковики могут обрабатывать их по-разному.

### «Можно ли стилизовать `<mark>` — изменить цвет фона?»

Да, через CSS:

```CSS
mark {
    background-color: lightblue;
    padding: 2px 4px;
    border-radius: 3px;
}
```

### «Как написать дробь, например ½?»

Есть несколько способов:

|Способ|Код|Результат|
|---|---|---|
|HTML-сущность|`&frac12;`|½|
|Sup + sub|`<sup>1</sup>/<sub>2</sub>`|¹⁄₂|
|Unicode-символ|`½` (скопировать)|½|

### «Можно ли вкладывать `<mark>` в `<strong>`?»

Да, теги форматирования свободно комбинируются:

```HTML
<p><strong><mark>Очень важный и выделенный текст</mark></strong></p>
```

### «Тег `<u>` устарел?»

Нет, в HTML5 `<u>` **не устарел** (не deprecated). Но его назначение сузилось — теперь он обозначает «текст, стилистически отличающийся от обычного» (например, намеренные ошибки, имена собственные в китайском). Для простого подчёркивания используй CSS.

## 🔑 Ключевые мысли урока

> [!abstract] Запомни
> 
> 1. **`<mark>`** — маркерное выделение (жёлтый фон), для контекстно-релевантного текста
> 2. **`<del>`** — удалённый текст (зачёркнутый), используется с `<ins>` для исправлений
> 3. **`<ins>`** — добавленный текст (подчёркнутый), пара к `<del>`
> 4. **`<s>`** — неактуальный текст (зачёркнутый), но **не удалённый**
> 5. **`<sub>`** — подстрочный текст (H₂O), для формул и индексов
> 6. **`<sup>`** — надстрочный текст (x²), для степеней и сносок
> 7. **`<small>`** — побочный текст (копирайт, оговорки), не для дизайна
> 8. **`<u>`** — подчёркивание, использовать **осторожно** (путают со ссылкой)
> 9. **`<abbr>`** — аббревиатура, обязательно с атрибутом `title`
